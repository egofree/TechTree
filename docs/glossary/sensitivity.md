# Sensitivity

> **Type**: equipment | **Tier**: important | **Domains**: chemistry, computing

NG is sensitive to impact (2 J initiation energy), friction, and rapid heating. It can be initiated by a hammer blow on a hard surface. Liquid NG in a glass container can detonate if the container is dropped. Dynamite manufacture absorbs NG into a solid matrix specifically to reduce this sensitivity.
