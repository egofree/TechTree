# Synchronous motors

> **Type**: noun | **Tier**: important | **Domains**: computing

Synchronous motors (constant speed) — Principle**: AC motor whose rotor rotation is synchronized with the AC supply frequency. At 50 Hz supply, a 2-pole synchronous motor runs at exactly 3,000 RPM; at 60 Hz, 3,600 RPM. Speed does not vary with load (up to the pull-out torque limit). - **Applications**: Clocks, timer mechanisms, chart recorders, phonograph turntables, telescopic drive motors. Any application requiring precise, constant speed tied to the power line frequency. See: [Electromechanical Computing](../computing/electromechanical.md)
