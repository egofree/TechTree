# Refrigerated storage

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Refrigerated storage (low temperature, near-atmospheric pressure) — Insulated steel tanks at -33°C, slightly above atmospheric pressure. Double-wall construction with insulated annular space. Tank size: 10,000-50,000 tonnes. External refrigeration system (compressor + condenser) maintains temperature by condensing boil-off vapor. Used for large-scale storage and import/export terminals.
