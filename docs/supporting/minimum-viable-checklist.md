# Minimum Viable Civilization Checklist

This combines the core path with the most helpful side quests into tiers. Focus on completing each tier before heavily investing in the next.

## Tier 0: Survival and Foundations (Years 0–5)

- [ ] Fire-making (friction, later flint-and-steel)
- [ ] Charcoal production (controlled pyrolysis)
- [ ] Basic stone and wood tools
- [ ] Agriculture surplus established
- [ ] Animal domestication for labor
- [ ] Basic mining and prospecting (copper, iron, quartz, limestone, clay, sulfur)
- [ ] Pottery and kilns
- [ ] Writing system developed (SQ1)
- [ ] Standardized basic measurements (SQ2)
- [ ] Sanitation and basic health practices (SQ5)
- [ ] Basic shelters and organization

## Tier 1: Industrial Base (Years 5–20)

- [ ] Copper smelting and bronze alloying
- [ ] Iron bloomery smelting → wrought iron → steel
- [ ] Basic metalworking tools (anvils, hammers, files, tongs)
- [ ] Water/wind power harnessing
- [ ] Basic glass production
- [ ] Machine tools bootstrap — charcoal foundry operational (Phase 3)
- [ ] First crude lathe built and iteratively improved
- [ ] Precision metrology expanding (SQ2)
- [ ] Steam power developed (Phase 4)
- [ ] Electricity generation — first generators operational
- [ ] Bulk chemistry beginning (Phase 5) — sulfuric acid production
- [ ] Transportation and logistics network (SQ3)
- [ ] Mechanical computing aids available (SQ4) — slide rules, calculators
- [ ] Advanced ceramics and refractories (SQ8)
- [ ] Printing press operational (SQ3)
- [ ] Roads and basic transport infrastructure

## Tier 2: Semiconductor Capability (Years 15–40)

- [ ] Full machine shop operational (shaper, mill, drill press, grinder)
- [ ] Reliable electricity generation and distribution
- [ ] Chemical industry producing: H₂SO₄, HCl, HNO₃, HF, NaOH, Na₂CO₃
- [ ] Electrolysis producing: Cl₂, H₂, O₂, NaOH
- [ ] Vacuum pumps and chambers operational
- [ ] Glass apparatus and basic optics
- [ ] Metallurgical-grade silicon produced in arc furnace
- [ ] Silicon purification pathway operational (directional solidification or chemical)
- [ ] Czochralski crystal growth producing single-crystal ingots
- [ ] Wafer slicing, lapping, and polishing
- [ ] First solar cells fabricated and generating power
- [ ] Basic diodes and transistors working
- [ ] Specialty gases supply chain (SQ6) — argon, nitrogen, hydrogen
- [ ] Basic cleanroom or glovebox capability
- [ ] Energy storage — lead-acid battery banks (SQ7)
- [ ] Knowledge/education system scaling (SQ1) — trade schools, libraries

## Tier 3: Advanced and Self-Sustaining (Years 30–100+)

- [ ] Photolithography capability — masks, exposure, alignment
- [ ] Photoresist production (novolac + sensitizer)
- [ ] Core fab processes: oxidation, etching, CVD, doping, metallization
- [ ] First integrated circuits (SSI/MSI)
- [ ] Wafer packaging, wire bonding, encapsulation
- [ ] Wafer probing and parametric testing
- [ ] Early computers/controllers operational from own ICs
- [ ] Cleanrooms at laminar flow level with UPW
- [ ] VLSI scaling in progress — smaller features, more layers
- [ ] Advanced lithography scaling (DUV development)
- [ ] High-end solar cells (PERC-like) in production
- [ ] Nuclear power research underway (SQ7) — long-term baseload
- [ ] Full computing ecosystem + EDA tools
- [ ] Comprehensive recycling and circular systems
- [ ] Specialized workforce training institutions (universities)
- [ ] Advanced packaging (flip-chip, chiplets) in development
- [ ] GPU-scale VLSI design capability

## Success Criteria for "Minimum Viable"

- Can produce usable solar cells and basic ICs
- Has redundant energy, transportation, and knowledge systems
- Can train the next generation of specialists
- Has positive feedback loops (early chips improve tools and processes)
