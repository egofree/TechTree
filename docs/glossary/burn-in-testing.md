# Burn-in testing

> **Type**: equipment | **Tier**: important | **Domains**: vlsi-scaling

Accelerated life screening that operates chips at elevated temperature (125-150°C) and elevated voltage (1.2-1.5× nominal) for 4-48 hours. Infant mortality defects (weak gate oxides, marginal via contacts, contamination) fail during burn-in rather than in the field. Burn-in is expensive (dedicated ovens, powered boards, handling equipment) but necessary for high-reliability applications. Monitoring burn-in current per chip detects failing devices without running functional tests.
