# Saponification value (SV)

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Milligrams of KOH required to saponify 1 gram of fat. Coconut oil: SV 250-260. Olive oil: SV 185-196. Tallow: SV 190-200. Palm oil: SV 195-205. These values determine exact lye quantities for complete reaction.
