# Gear cutting

> **Type**: noun | **Tier**: important | **Domains**: computing

Involute tooth profile generated by hobbing (rotating cutter synchronized with gear blank rotation) or milling with formed cutter. Module (tooth size) = pitch diameter / number of teeth. Typical calculator gears: module 0.5-1.0 mm, 20-60 teeth. Tooth spacing tolerance: ±0.01 mm. Backlash (play between meshing gears): <0.02 mm for carry mechanism reliability.
