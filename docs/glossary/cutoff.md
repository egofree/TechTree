# Cutoff

> **Type**: noun | **Tier**: important | **Domains**: silicon

(base-emitter junction not forward biased: Ic ≈ 0). **Active** (base-emitter forward biased, base-collector reverse biased: Ic = β·Ib, amplifier operation). **Saturation** (both junctions forward biased: Vce ≈ 0.2V, switch "on" state). **Reverse active** (junctions reversed, β very low, rarely used).
