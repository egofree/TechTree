# Konrad Zuse Z3

> **Type**: noun | **Tier**: important | **Domains**: computing

(1941, first working programmable computer): - **Architecture**: 64 words of 22-bit floating-point memory (relays), arithmetic unit using relay logic, program on punched 35 mm film strip. Binary floating-point representation: 1 sign bit, 7 exponent bits, 14 mantissa bits. - **Hardware**: ~2,000 relays total. Clock speed 4-5 Hz. Addition: 3 cycles (~0.7 seconds).
