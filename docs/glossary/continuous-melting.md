# Continuous melting

> **Type**: process | **Tier**: important | **Domains**: glass

Continuous melting (production method for optical glass) — - Raw materials batched to ±0.01% accuracy (vs. ±2% for container glass). Weighing errors shift the refractive index outside specification. - Melt in a platinum-lined tank furnace. Platinum is inert and introduces no contamination, unlike clay crucibles which dissolve slowly into the glass.
