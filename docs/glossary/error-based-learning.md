# Error-based learning

> **Type**: noun | **Tier**: important | **Domains**: knowledge

When the apprentice makes a mistake (and they will), the master uses it as a teaching moment: "You see this? This happened because [root cause]. Next time, [prevention]. Now, [correction]." Errors are expected, not punished — provided the apprentice learns from them. The apprentice's notebook should record every error, its cause, and its remedy. This error log becomes a personal troubleshooting manual.
