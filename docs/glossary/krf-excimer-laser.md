# KrF excimer laser

> **Type**: material | **Tier**: important | **Domains**: vlsi-scaling

KrF excimer laser (248 nm) — Features down to ~0.25 μm. Laser gas: Kr + F₂ mixture, electrically pumped. 1000 Hz pulse rate, ~10 W output. See: [Advanced Lithography](../vlsi-scaling/advanced-lithography.md)
