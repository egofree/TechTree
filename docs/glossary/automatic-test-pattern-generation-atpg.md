# Automatic test pattern generation (ATPG)

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Algorithms generate the minimum set of test patterns to detect all manufacturing defects. Stuck-at fault model: each net is assumed to be stuck at 0 or stuck at 1. ATPG generates patterns that propagate each fault to an observable output. Transition fault model: tests for slow-to-rise and slow-to-fall defects (timing-related failures). Fault coverage target: >99% for stuck-at, >95% for transition.
