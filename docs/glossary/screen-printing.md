# Screen printing

> **Type**: noun | **Tier**: important | **Domains**: knowledge, silicon

(serigraphy): Stretch a fine mesh screen (silk historically, now polyester or nylon, 80-200 threads per inch depending on detail required) tightly over a wooden or aluminum frame. Block areas of the screen that should not print using a stencil (paper, film, or photo-emulsion). Force ink through the open mesh areas onto the substrate below using a squeegee (rubber blade).
