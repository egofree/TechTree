# Packed batch column

> **Type**: material | **Tier**: important | **Domains**: chemistry

Packed batch column (Year 5-15) — Add 1-3 m ceramic Raschig rings or saddles. Reflux control via manual split of condensate. 5-10 theoretical plates. Solvent purification, coal tar light oil fractionation, ethanol to 85-90%. See: [Distillation](../chemistry/distillation.md)
