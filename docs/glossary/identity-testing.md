# Identity testing

> **Type**: material | **Tier**: important | **Domains**: chemistry, health

Confirm the product contains the correct active ingredient. Methods: FTIR spectroscopy (Fourier transform infrared, compares sample spectrum to reference standard), Raman spectroscopy (non-destructive, can test through glass or plastic packaging), HPLC retention time matching, or colorimetric reaction (specific to the drug molecule). Every batch must pass identity testing before release.
