# Mill configuration

> **Type**: noun | **Tier**: important | **Domains**: metals

Mill configuration: 10-25 stands in a continuous line (no reversing). Roughing group (4-6 stands, horizontal-vertical alternation) → intermediate group (4-8 stands) → finishing block (4-10 stands in a compact housing). Stands are increasingly smaller and faster. See: [Primary Metal Forming](../metals/forming.md)
