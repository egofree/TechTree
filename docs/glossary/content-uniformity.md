# Content uniformity

> **Type**: material | **Tier**: important | **Domains**: chemistry

Per USP <905>, test 10 dosage units individually for drug content. Acceptance: each unit contains 85-115% of label claim, and the relative standard deviation (RSD) is ≤6%. If one unit falls outside 85-115% but within 75-125%, test 20 more units. All 30 units must be 75-125%, and RSD of the 30 must be ≤7.8%.
