# Punch card

> **Type**: noun | **Tier**: important | **Domains**: computing

(for data and program storage): - **Card**: Stiff paper card (187 × 83 mm, the "IBM card" or "Hollerith card"). 12 rows × 80 columns. Each column represents one character. Punch holes in specific row positions to encode data (numeric 0-9, or alphanumeric via zone punches in rows 0, 11, 12 combined with digit punches). - **Card punch**: Keyboard-operated machine that punches holes in card.
