# Tungsten carbide synthesis

> **Type**: equipment | **Tier**: important | **Domains**: metals

Tungsten powder (1-5 µm) + carbon black (6.1-6.3% C by weight) → WC. Carburization at 1400-1600°C in H₂ atmosphere for 1-4 hours in pusher furnaces. Reaction: W + C → WC (ΔH = -35 kJ/mol, exothermic). Temperature control is critical: below 1350°C, incomplete carburization leaves W₂C (brittle eta-phase); above 1650°C, grain coarsening reduces hardness.
