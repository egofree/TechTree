# Maritime rigging

> **Type**: noun | **Tier**: important | **Domains**: textiles

(standing rigging, running rigging, mooring, anchor rode): Saltwater demands tarred hemp or manila. Standing rigging (shrouds, stays) carries constant load — use thick rope, inspect for chafe at deadeyes and pins. Running rigging (sheets, halyards) passes through blocks — must be flexible, choose manila or lighter hemp.
