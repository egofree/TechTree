# Architecture

> **Type**: noun | **Tier**: important | **Domains**: computing

Architecture: 20 accumulators (each a 10-digit decimal counter using ring circuits), multiplier, divider/square-root unit, function tables (read-only lookup via switches), master clock (100 kHz ring oscillator). All decimal, not binary. See: [Electromechanical Computing](../computing/electromechanical.md) | [Electronic Computing](../computing/electronic.md)
