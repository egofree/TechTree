# Vernier caliper

> **Type**: noun | **Tier**: important | **Domains**: measurement

Two scales — main scale (mm divisions) and vernier scale (sliding, 49 mm divided into 50 divisions → each vernier division = 0.98 mm, difference = 0.02 mm). Reading: align zero of vernier with reading on main scale, find which vernier line aligns with a main scale line → add vernier value to main scale reading. Resolution 0.02 mm. Constructed from hardened stainless steel.
