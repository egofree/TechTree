# Slag

> **Type**: material | **Tier**: important | **Domains**: silicon

Slag: 200-400 kg per tonne of silicon. Non-hazardous if heavy metals are below regulatory limits. Can be crushed and used as construction aggregate, road base, or cement additive. Test for leachable heavy metals before any land application. See: [Metallurgical-Grade Silicon Production](../silicon/mg-si-production.md)
