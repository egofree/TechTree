# SEM Tech

> **Type**: process | **Tier**: critical | **Domains**: chemistry

Salt Electro Mining Technology — an open-source electrochemical platform developed by Robert Karas (Rowow LLC) that uses saltwater and electricity to generate leaching chemicals (HCl + sodium chlorate) in-situ and extract metals. Uses low-cost ion exchange membranes made from pulverized pre-functionalized resin beads in PVC/CPVC matrix. Patent application (not granted), CC0 1.0 Universal license.
