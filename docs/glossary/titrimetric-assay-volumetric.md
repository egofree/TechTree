# Titrimetric assay (volumetric)

> **Type**: material | **Tier**: important | **Domains**: mining

Dissolve sample, react the target element with a standard solution of known concentration until the endpoint is reached (indicated by color change). Iodometric titration for copper: dissolve in acid, add KI, titrate liberated iodine with sodium thiosulfate (Na₂S₂O₃) using starch indicator (blue → colorless endpoint). Each mL of 0.1 N Na₂S₂O₃ = 6.35 mg Cu. Accuracy ±0.2% for copper concentrations >1%.
