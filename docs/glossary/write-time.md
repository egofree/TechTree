# Write time

> **Type**: noun | **Tier**: important | **Domains**: photolithography

Write time refers to 1-8 hours per reticle, depending on pattern density and spot size. Dense patterns (memory chips) take longer than sparse patterns (analog circuits). The e-beam writes one pixel at a time, so write time scales with total pattern area. See: [Photoresists, Masks & Lithography](../photolithography/resists-masks.md)
