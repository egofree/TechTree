# Powder formulation

> **Type**: material | **Tier**: important | **Domains**: chemistry

Thermosetting powder coatings contain: resin (epoxy, polyester, or hybrid, 50-60% by weight), curing agent (blocked isocyanate, TGIC, or primid, 3-5%), pigment (TiO₂, iron oxide, carbon black, 20-40%), flow agent (polyacrylate, 0.5-2%), and degassing agent (benzoin, 0.3-0.5%).
