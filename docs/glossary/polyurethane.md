# Polyurethane

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Isocyanate + polyol reaction. Two-component for high-performance coatings (automotive, aircraft, industrial flooring). Excellent abrasion resistance, gloss retention, UV resistance (aliphatic isocyanate types like HDI biuret). One-component moisture-cure urethanes react with atmospheric moisture — used for floor and marine coatings.
