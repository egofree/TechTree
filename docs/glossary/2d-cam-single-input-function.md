# 2D cam (single-input function)

> **Type**: noun | **Tier**: important | **Domains**: computing

For y = f(x), the cam rotates proportional to x. A follower rides the cam profile, its displacement proportional to y. The cam profile is machined to produce the desired displacement at each angular position. A sine-function cam is simply a circle mounted eccentrically (off-center). A logarithmic cam requires a specially computed profile.
