# Technical drawings

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Standardized format: - Title block (part name, drawing number, scale, date, drawn-by, material, tolerances) - Multiple views (plan/top, elevation/front, section/cross-section) - Dimensions with tolerances (e.g., 25.00 ±0.05 mm) - Surface finish symbols - Material specifications - Assembly instructions where applicable
