# Handling

> **Type**: noun | **Tier**: supporting | **Domains**: gas-handling, textiles

Never touch vacuum-facing surfaces with bare hands. Skin oils (sebum, primarily fatty acids and squalene) have extremely low vapor pressure and persist on surfaces for days. Use clean nitrile gloves (powder-free) when handling vacuum components, flanges, gaskets, and fixtures. Change gloves if they contact a non-clean surface (bench top, tool handle, clothing). Handle components by their exterior (atmospheric) surfaces whenever possible.
