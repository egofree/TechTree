# Post-and-beam

> **Type**: material | **Tier**: important | **Domains**: foundations

Vertical posts (15-20 cm diameter) set in ground holes 60-90 cm deep. Horizontal beams notched into post tops. Stone or bronze tools sufficient for all joinery. Spacing 1.5-2.5 m between posts. Pack stones around the base of each post in the hole for drainage, then backfill with tamped earth. Treat buried post ends with char (light charring over fire) to slow rot. A properly built post-and-beam frame lasts 20-40 years before post bases need replacement.
