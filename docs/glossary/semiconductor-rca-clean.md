# Semiconductor RCA clean

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Standard wafer cleaning sequence uses H₂O₂ as a critical reagent: - **SC-1** (Standard Clean 1): NH₄OH:H₂O₂:H₂O = 1:1:5 at 75-80°C. Removes organic contamination and particles. H₂O₂ oxidizes surface silicon to SiO₂, NH₄OH provides particle repulsion via zeta potential. - **SC-2** (Standard Clean 2): HCl:H₂O₂:H₂O = 1:1:6 at 75-80°C. Removes metallic contamination.
