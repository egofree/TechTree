# Functional simulation (RTL)

> **Type**: material | **Tier**: important | **Domains**: vlsi-scaling

Simulates the behavioral HDL at zero delay (no timing). Fast enough for full-chip tests (billions of clock cycles). Testbenches drive stimulus into the design and check outputs against expected results. Coverage metrics track what fraction of design states and transitions have been exercised — target >95% code coverage and >90% functional coverage before tapeout.
