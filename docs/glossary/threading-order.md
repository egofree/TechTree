# Threading order

> **Type**: noun | **Tier**: important | **Domains**: textiles

Transfer warp to loom. Thread each warp end through a heddle (one per shaft) in the pattern sequence — for plain weave: shaft 1, shaft 2, repeat. For twill: shaft 1, 2, 3, 4, repeat. Sley through reed (determine ends per inch / dent spacing) for proper cloth width.
