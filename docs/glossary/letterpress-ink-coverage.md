# Letterpress ink coverage

> **Type**: material | **Tier**: important | **Domains**: knowledge

Letterpress ink coverage refers to 0.5-1.5 g/m² per color per impression, depending on type density (a full page of solid text uses more ink than a sparse page). An 80 gsm A4 page with average text coverage consumes ~0.3-0.4 g of ink per side. See: [Paper and Printing Press](../knowledge/printing.md)
