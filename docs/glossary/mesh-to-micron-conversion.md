# Mesh-to-micron conversion

> **Type**: noun | **Tier**: important | **Domains**: machine-tools

US mesh number ≈ 15,000 / particle diameter in μm (approximate). For example, 60 mesh ≈ 250 μm, 120 mesh ≈ 125 μm, 220 mesh ≈ 63 μm. The conversion is not exact because particles are irregular, not spherical, and each grit grade spans a size distribution rather than a single dimension.
