# Flint glass

> **Type**: material | **Tier**: important | **Domains**: glass

(high dispersion): Lead glass base, RI 1.58-1.72, Abbe number 25-40. Used for the concave (negative) element in achromatic doublets. The high RI allows a shallower curve for the same optical power, reducing spherical aberration. The combination of a crown convex and flint concave element, cemented together, produces an achromatic lens that focuses two wavelengths to the same point.
