# Sexagesimal system

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Sexagesimal system (Sumerian/Babylonian, ~3000 BCE) — Base-60 number system. 60 is divisible by 2, 3, 4, 5, 6, 10, 12, 15, 20, 30 — convenient for fractions. Survives today in 60-second minutes, 60-minute hours, 360-degree circles. Written with two cuneiform signs: a vertical wedge for 1, a sideways wedge for 10. Place-value notation (position determines magnitude) — the earliest known positional number system. Lacked a zero symbol, causing ambiguity (e.g., "1 3" could be 1×60 + 3 = 63 or 1×3600 + 3 = 3603).
