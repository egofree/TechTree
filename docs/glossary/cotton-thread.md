# Cotton thread

> **Type**: noun | **Tier**: supporting | **Domains**: textiles

Sewing thread made from spun cotton fibers, typically 2-3 ply mercerized for strength and luster. Thread size numbered by tex (grams per 1000 m) or ticket number; common sewing thread is tex 25-40. Mercerization (NaOH treatment) increases strength by 10-20% and improves dye uptake.
