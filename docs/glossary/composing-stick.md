# Composing stick

> **Type**: noun | **Tier**: important | **Domains**: knowledge

A metal tray with an adjustable side wall (sets line length). The compositor selects type pieces from the type case with one hand (the other hand holds the composing stick), reading the manuscript text. Each type piece is placed upside-down and backwards in the stick — the type reads right when printed, so the compositor must read it in mirror image. Fill the stick line by line, inserting spaces (quadrats — blank type pieces of varying widths) for word spacing and justification.
