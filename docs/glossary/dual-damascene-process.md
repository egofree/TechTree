# Dual damascene process

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Dual damascene process (7 steps per metal layer) — 1. **Dielectric deposition**: Deposit inter-layer dielectric (SiO₂ or low-k material). CVD or spin-on. 500-1000 nm thick. 2. **Lithography + etch**: Pattern trenches (wires) and vias (vertical connections) in the dielectric using two lithography steps + RIE. Dual damascene patterns both in one etch sequence. 3. **Barrier/seed deposition**: Deposit Ta/TaN barrier layer (5-20 nm by PVD or ALD) to prevent Cu diffusion into SiO₂. Deposit Cu seed layer (50-100 nm by PVD)... See: [Advanced Processes](../vlsi-scaling/advanced-processes.md)
