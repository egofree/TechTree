# Signal routing

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Connect all signal nets (cell outputs to cell inputs) using metal layers. Manhattan routing (horizontal and vertical only) on a grid. Global routing assigns nets to routing channels; detailed routing creates actual wire geometries. Design rules enforce minimum wire width, spacing, and via enclosure.
