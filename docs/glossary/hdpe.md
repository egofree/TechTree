# HDPE

> **Type**: noun | **Tier**: important | **Domains**: polymers

HDPE (high-density) — low-pressure Ziegler-Natta process. 1-20 atm, 50-80°C, TiCl₄/AlEt₃ catalyst system. Linear chain structure yields stiffer, stronger material with higher crystallinity. See: [Thermoplastic Polymers](../polymers/thermoplastics.md)
