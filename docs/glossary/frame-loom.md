# Frame loom

> **Type**: noun | **Tier**: important | **Domains**: textiles

Frame loom (simplest) — - **Construction**: Rectangular wooden frame. Warp (lengthwise) threads tied top and bottom. Weft (crosswise) thread passed over and under warp by hand. Heddle bar (flat stick with holes and slots) lifts alternate warp threads to create shed (gap for weft passage). Width: 30-60 cm. Length limited by frame size. - **Production**: Very slow (~0.5-1 m²/hour), but requires minimal construction. See: [Weaving](../textiles/weaving.md)
