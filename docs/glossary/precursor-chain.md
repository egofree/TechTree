# Precursor chain

> **Type**: noun | **Tier**: important | **Domains**: polymers

MG-Si + CH₃Cl → dimethyl dichlorosilane via Rochow-Müller process (Cu catalyst, 250-300°C) → hydrolysis → cyclic/linear siloxanes → polydimethylsiloxane (PDMS). See [Rubber Production](./rubber.md) for detailed precursor chemistry.
