# Alkaline electrolysis (most practical for bootstrap)

> **Type**: noun | **Tier**: supporting | **Domains**: chemistry

Water splitting using a potassium hydroxide (KOH) electrolyte between nickel-coated electrodes, producing hydrogen and oxygen. Preferred for bootstrap scenarios because it uses non-precious-metal catalysts, tolerates less-pure feed water, and operates at moderate temperatures (60-80°C).
