# Transistor-transistor logic (TTL)

> **Type**: noun | **Tier**: important | **Domains**: computing

Multi-emitter input transistor implements AND function, output stage provides inversion and drive. Standard 5 V supply. Propagation delay: 10-30 ns per gate. Power: ~10 mW per gate. Fan-out: 10 (one gate output drives 10 gate inputs). The workhorse logic family of the 1960s-1970s.
