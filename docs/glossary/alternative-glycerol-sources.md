# Alternative glycerol sources

> **Type**: noun | **Tier**: important | **Domains**: chemistry

(post-petroleum): - **Hydrolysis of fats (not saponification)**: Fat + water (at 250-260°C, 40-50 bar, in an autoclave) → fatty acids + glycerol. No alkali consumed. The "Twitchell process" uses sulfuric acid + aromatic sulfonic acid catalyst. The "Colgate-Emery process" uses continuous high-pressure hydrolysis without catalyst.
