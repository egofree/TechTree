# Compounding

> **Type**: noun | **Tier**: important | **Domains**: polymers

— blend on mill or in Banbury mixer with: - Sulfur (1.5-3 phr) — crosslinking agent - Accelerators (e.g., mercaptobenzothiazole, 0.5-1.5 phr) - Zinc oxide (3-5 phr) — activator - Stearic acid (1-2 phr) — co-activator - Carbon black (20-50 phr) — reinforcement - Antioxidants (1-2 phr)
