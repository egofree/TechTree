# Discharge

> **Type**: noun | **Tier**: important | **Domains**: energy

Pb (anode) + PbO₂ (cathode) + 2H₂SO₄ → 2PbSO₄ + 2H₂O (cell voltage: 2.05V nominal) - **Charge**: Reverse reaction — apply external voltage >2.4V per cell → convert PbSO₄ back to Pb and PbO₂, regenerate H₂SO₄. - **Electrolyte**: 30-40% sulfuric acid (H₂SO₄) in water. Density indicates state of charge: fully charged = 1.265 g/cm³, discharged = 1.120 g/cm³. Measure with hydrometer (glass float in electrolyte sample — calibrated markings indicate density/charge state).
