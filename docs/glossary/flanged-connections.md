# Flanged connections

> **Type**: noun | **Tier**: important | **Domains**: gas-handling

Flat-faced or raised-face flanges bolted together with gasket between. Bolt pattern (number and circle diameter) standardized by pressure class (e.g., 4 bolts for 1" Class 150, 8 bolts for 4" Class 150). Tighten bolts in crisscross pattern to seat gasket evenly. Allow re-torque after first thermal cycle.
