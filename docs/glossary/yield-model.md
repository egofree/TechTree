# Yield model

> **Type**: noun | **Tier**: important | **Domains**: photolithography

Yield = (1 - D·A)ⁿ, where D = defect density (defects/cm² per layer), A = die area (cm²), n = number of critical lithography layers. For a die of 0.5 cm² with 7 critical layers and a defect density of 0.5 defects/cm² per layer: Yield = (1 - 0.5 × 0.5)⁷ = (0.75)⁷ = 0.133 = 13.3%. Reducing defect density to 0.1/cm² per layer: Yield = (0.95)⁷ = 0.698 = 69.8%. This exponential relationship between defect density and yield is why cleanroom discipline and contamination control are so important.
