# Ball-and-disk integrator

> **Type**: noun | **Tier**: important | **Domains**: computing

A steel ball sits between a rotating disk and a cylindrical output roller. The ball contacts the disk at a radial position set by an input shaft. The disk rotates at constant speed. The ball transmits rotation to the output roller at a rate proportional to the ball's radial position (distance from disk center). Moving the ball outward increases the output roller speed.
