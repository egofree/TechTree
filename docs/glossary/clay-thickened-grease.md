# Clay-thickened grease

> **Type**: noun | **Tier**: important | **Domains**: chemistry

(non-soap): - Bentonite clay (organically modified — treated with quaternary ammonium salts to make it oil-compatible) + base oil. No dropping point (clay doesn't melt) → usable to 250°C+. For high-temperature applications where soap grease fails. Simpler to make than lithium grease if bentonite clay is available.
