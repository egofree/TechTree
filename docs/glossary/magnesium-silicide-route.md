# Magnesium silicide route

> **Type**: noun | **Tier**: important | **Domains**: chemistry

(laboratory): Mg₂Si + 4HCl → 2MgCl₂ + SiH₄. Produces impure silane with phosphine (PH₃) and arsine (AsH₃) contaminants — dangerous and unsuitable for semiconductor-grade material. Only for small-scale/non-electronic applications.
