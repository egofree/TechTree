# Typical parameters

> **Type**: material | **Tier**: important | **Domains**: photolithography

1000 RPM (thick coat, 3-5 μm), 3000 RPM (standard, 1.0-1.5 μm), 6000 RPM (thin coat, 0.5-0.8 μm). Ramp rate: 1000-5000 RPM/s acceleration. Slow ramp produces more uniform films but may allow edge bead buildup.
