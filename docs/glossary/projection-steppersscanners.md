# Projection steppers/scanners

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

(features down to ~0.5 μm): - **Step-and-repeat**: Lens projects reduced mask image (typically 5:1 or 10:1 reduction) onto one exposure field (~20×20 mm). Stage steps to next field. Each wafer requires 50-200+ exposures. - **Alignment**: Automatic alignment using microscope + pattern recognition. Overlay accuracy: ±0.1-0.5 μm. - **I-line (365 nm)**: Mercury lamp + narrowband filter.
