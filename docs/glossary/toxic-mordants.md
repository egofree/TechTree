# Toxic mordants

> **Type**: noun | **Tier**: important | **Domains**: textiles

Alum (aluminum potassium sulfate) is relatively safe. Iron sulfate is a mild irritant. Copper sulfate is toxic if ingested. Chrome mordant (potassium dichromate, K₂Cr₂O₇) is a known carcinogen and must be handled with gloves. Tin chloride is toxic. Wear gloves when handling mordants.
