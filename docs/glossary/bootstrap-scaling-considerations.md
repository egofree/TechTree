# Bootstrap scaling considerations

> **Type**: equipment | **Tier**: important | **Domains**: silicon

A minimal silicon production facility for bootstrapping semiconductor capability starts with a single 5 MWA furnace producing ~2,000 tonnes MG-Si per year. This provides enough silicon to feed a small polysilicon plant (producing ~500 tonnes/year of semiconductor-grade polysilicon), which feeds a CZ crystal puller producing ~200 tonnes/year of single crystal ingots.
