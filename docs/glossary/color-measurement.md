# Color measurement

> **Type**: noun | **Tier**: important | **Domains**: chemistry

CIELAB color space (L* = lightness, a* = red-green, b* = yellow-blue) quantifies color objectively. Spectrophotometer measures reflectance at 10 nm intervals across 400-700 nm. Color difference ΔE = √((ΔL*)² + (Δa*)² + (Δb*)²). A ΔE < 1.0 is imperceptible to most observers; ΔE 1-3 is a slight mismatch; ΔE > 3 is a visible color difference.
