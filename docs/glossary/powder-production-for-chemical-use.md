# Powder production for chemical use

> **Type**: equipment | **Tier**: important | **Domains**: silicon

Jaw crush primary ingot chunks to <50 mm, then ball mill with alumina grinding media to <1 mm. Air classify to target size distribution: the classifier's adjustable rotor speed sets the cutoff between fines and product. Typical chemical-grade silicon powder: 0.1-5 mm particle size, bulk density 1.0-1.5 g/cm³.
