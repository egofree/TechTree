# Transmission line design

> **Type**: material | **Tier**: important | **Domains**: energy

Conductor material is almost always aluminum conductor steel reinforced (ACSR), where aluminum strands carry the current and a steel core provides tensile strength. Copper is too heavy and expensive for long spans. Conductor size is specified by cross-sectional area (25-400 mm² aluminum). Larger conductors carry more current with less loss but cost more and require stronger towers.
