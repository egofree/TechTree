# Cell placement

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Standard cells placed in rows within each block. Each cell occupies a fixed-height row slot (cell height is uniform across the library, typically 10-15 tracks). Simulated annealing or analytical placement algorithms minimize half-perimeter wirelength (HPWL) while respecting congestion constraints and timing requirements. Placement density typically 60-80% to leave routing channels.
