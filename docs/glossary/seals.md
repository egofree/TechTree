# Seals

> **Type**: material | **Tier**: important | **Domains**: polymers

O-rings (circular cross-section, Shore hardness A40-A90, tolerances ±0.10-0.15 mm on cross-section diameter). Shaft seals (lip seals with spring-loaded rubber lip running on rotating shaft). Gaskets (flat or molded profiles, static sealing). Hardness selection depends on application — soft (A40-60) for low-pressure conformability, hard (A70-90) for extrusion resistance in high-pressure service.
