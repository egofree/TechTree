# Plated wire memory

> **Type**: material | **Tier**: important | **Domains**: computing

(later improvement): - **Structure**: Thin copper wire (0.1 mm diameter) electroplated with permalloy (Ni-Fe alloy, ~1 μm thick). Wires woven through ferrite keeper planes. Magnetic domains along the wire store bits.
