# Grading system

> **Type**: noun | **Tier**: important | **Domains**: textiles

Assign cloth to quality grades based on defect count per 100 linear meters. Grade A: 0-3 major defects (functional impairment). Grade B: 4-8 major defects. Grade C: 8+ major defects — usable only for low-grade applications. Penalties assessed on a point system — 1 point per defect, 4 points maximum per linear yard regardless of defect count.
