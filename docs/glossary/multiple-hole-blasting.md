# Multiple-hole blasting

> **Type**: material | **Tier**: important | **Domains**: mining

For larger excavations, drill multiple holes in a pattern (grid or line). Space holes at 1-1.5× the burden distance. Connect all fuses to a single slow match for simultaneous initiation. Simultaneous blasts are more effective than sequential shots because the rock is already fractured by neighboring charges, creating additional free faces for each charge to work against.
