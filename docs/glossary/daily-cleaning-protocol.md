# Daily cleaning protocol

> **Type**: material | **Tier**: important | **Domains**: photolithography

Wipe all work surfaces, equipment handles, and keyboard covers with 70% IPA (isopropyl alcohol) dampened lint-free wipes at the start and end of each shift. IPA evaporates quickly without residue. Replace sticky mats at entrances when tack diminishes. Empty and wipe process tool catch trays. Inspect and clean wafer carrier interiors before each use.
