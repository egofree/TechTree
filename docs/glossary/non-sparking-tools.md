# Non-sparking tools

> **Type**: material | **Tier**: important | **Domains**: chemistry, mining

Bronze or beryllium copper tools (wrenches, hammers, scrapers) for all work with explosives and in magazines. Steel-on-steel impacts generate sparks capable of initiating sensitive primary explosives and black powder. Aluminum-bronze alloy (95% Cu, 5% Al) provides hardness without spark generation.
