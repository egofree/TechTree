# Egyptian hieratic numerals

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Egyptian hieratic numerals (~3000 BCE) — base-10, additive notation (each symbol repeated — III = 3). Separate symbols for 1, 10, 100, 1000, 10,000, 100,000. Simple for recording quantities, cumbersome for calculation (no place value). See: [Writing & Record-Keeping](../knowledge/writing.md)
