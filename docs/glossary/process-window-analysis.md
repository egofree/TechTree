# Process window analysis

> **Type**: noun | **Tier**: important | **Domains**: photolithography

Vary exposure dose and focus on a test reticle containing dense, isolated, and minimum-size features. For each feature type, determine the dose/focus region where CD is within specification. The intersection of all individual process windows is the common process window. If the common window is too small, process adjustments are needed (different NA, illumination mode, or resist process).
