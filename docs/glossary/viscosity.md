# Viscosity

> **Type**: noun | **Tier**: important | **Domains**: chemistry, knowledge

Change >10% indicates oxidation, contamination, or wrong oil grade. (2) **Acid number**: Increase indicates oxidation — acidic byproducts corrode bearings. (3) **Particle count**: ISO 4406 cleanliness code — critical for hydraulic systems (target: 16/14/11 for servo valves). (4) **Spectrometric analysis**: ICP or atomic absorption detects wear metals (Fe, Cu, Pb, Sn, Cr) at ppm levels — trending indicates which component is wearing.
