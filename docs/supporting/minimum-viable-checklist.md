# Minimum Viable Civilization Checklist

This combines the core path with the most helpful parallel capabilities into tiers. Focus on completing each tier before heavily investing in the next.

## Tier 0: Survival and Foundations (Years 0‚Äì5)

- [ ] Fire-making (`foundations.fire`) ‚Äî friction, later flint-and-steel
- [ ] Charcoal production (`energy.fuels.charcoal`) ‚Äî controlled pyrolysis
- [ ] Basic stone and wood tools (`foundations.tools-basic`)
- [ ] Agriculture surplus established (`foundations.food-agriculture`)
- [ ] Animal domestication for labor (`foundations.food-agriculture`)
- [ ] Basic mining and prospecting (`mining.prospecting`) ‚Äî copper, iron, quartz, limestone, clay, sulfur
- [ ] Pottery and kilns (`ceramics.pottery`)
- [ ] Writing system developed (`knowledge.writing`)
- [ ] Standardized basic measurements (`measurement.precision-metrology`)
- [ ] Sanitation and basic health practices (`health.sanitation`)
- [ ] Basic shelters and organization
- [ ] Basic cordage and rope from natural fibers (`textiles`)

## Tier 1: Industrial Base (Years 5‚Äì20)

- [ ] Copper smelting and bronze alloying (`metals.copper-bronze`)
- [ ] Iron bloomery smelting ‚Üí wrought iron ‚Üí steel (`metals.iron-steel`)
- [ ] Basic metalworking tools (anvils, hammers, files, tongs)
- [ ] Water/wind power harnessing (`energy.gravity`, `energy.wind`)
- [ ] Basic glass production (`glass.basic`)
- [ ] Machine tools bootstrap ‚Äî charcoal foundry operational (`machine-tools.casting`)
- [ ] First crude lathe built and iteratively improved (`machine-tools.iterative-bootstrap`)
- [ ] Precision metrology expanding (`measurement.precision-metrology`)
- [ ] Steam power developed (`energy.steam-power`)
- [ ] Electricity generation ‚Äî first generators operational (`energy.electricity`)
- [ ] Bulk chemistry beginning (`chemistry.acids`) ‚Äî sulfuric acid production
- [ ] Transportation and logistics network (`transport`)
- [ ] Mechanical computing aids available (`computing.mechanical`) ‚Äî slide rules, calculators
- [ ] Advanced ceramics and refractories (`ceramics`)
- [ ] Printing press operational (`knowledge`)
- [ ] Roads and basic transport infrastructure (`transport.roads`)
- [ ] Lubrication with animal fats and vegetable oils established (`chemistry.lubricants`)
- [ ] Underground mining techniques developed ‚Äî ventilation, drainage (`mining`)
- [ ] Basic textile production ‚Äî spinning, weaving, canvas (`textiles`)

- [ ] Full machine shop operational ‚Äî shaper, mill, drill press, grinder (`machine-tools.iterative-bootstrap`)
- [ ] Reliable electricity generation and distribution (`energy.electricity`)
- [ ] Chemical industry producing: H‚ÇÇSO‚ÇÑ, HCl, HNO‚ÇÉ, HF, NaOH, Na‚ÇÇCO‚ÇÉ (`chemistry.acids`, `chemistry.alkalis`)

## Tier 3: Advanced and Self-Sustaining (Years 30‚Äì100+)

- [ ] Photolithography capability ‚Äî masks, exposure, alignment (`photolithography.resists-masks`)
- [ ] Photoresist production ‚Äî novolac + sensitizer (`photolithography.resists-masks`)
- [ ] Core fab processes: oxidation, etching, CVD, doping, metallization (`photolithography.fab-processes`)
- [ ] First integrated circuits (SSI/MSI) (`photolithography.fab-processes`)
- [ ] Wafer packaging, wire bonding, encapsulation (`chemistry.packaging-testing`)
- [ ] Wafer probing and parametric testing (`chemistry.packaging-testing`)
- [ ] Early computers/controllers operational from own ICs (`computing.electronic`)
- [ ] Cleanrooms at laminar flow level with UPW (`photolithography.cleanrooms`)
- [ ] VLSI scaling in progress ‚Äî smaller features, more layers (`vlsi-scaling.continuous-scaling`)
- [ ] Advanced lithography scaling ‚Äî DUV development (`vlsi-scaling.advanced-lithography`)
- [ ] High-end solar cells (PERC-like) in production (`vlsi-scaling.advanced-processes`)
- [ ] Full computing ecosystem + EDA tools (`vlsi-scaling.eda-design`)
- [ ] Comprehensive recycling and circular systems
- [ ] Specialized workforce training institutions (universities) (`knowledge.education`)
- [ ] Advanced packaging (flip-chip, chiplets) in development (`vlsi-scaling.eda-design`)
- [ ] GPU-scale VLSI design capability (`vlsi-scaling.eda-design`)

## Success Criteria for "Minimum Viable"

- Can produce usable solar cells and basic ICs
- Has redundant energy, transportation, and knowledge systems
- Can train the next generation of specialists
- Has positive feedback loops (early chips improve tools and processes)
