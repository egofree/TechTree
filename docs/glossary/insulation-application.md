# Insulation application

> **Type**: material | **Tier**: important | **Domains**: polymers

Gutta-percha was heated to 80-100°C in a steam-jacketed kettle to soften it into a homogeneous, bubble-free melt. The copper conductor was drawn horizontally through a pressurized gutta-percha extrusion die (similar to modern plastic wire-coating extrusion but with hand-fed gutta-percha). Multiple passes built up the insulation wall thickness: first pass ~1 mm, second pass to ~2.5 mm, third pass to final 3-5 mm. Each pass was cooled in a water trough before the next.
