# Nickel plating

> **Type**: noun | **Tier**: important | **Domains**: metals

Nickel plating (Watts bath) — Bath composition: NiSO₄·6H₂O (250-350 g/L) + NiCl₂·6H₂O (40-60 g/L) + H₃BO₃ (30-45 g/L, buffer) + organic brighteners (saccharin, allyl sulfonate, butynediol). Temperature: 50-65°C, pH 3.5-4.5, current density 20-50 mA/cm². - Produces a dense, ductile deposit with excellent corrosion resistance.
