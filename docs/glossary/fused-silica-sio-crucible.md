# Fused silica (SiO₂) crucible

> **Type**: noun | **Tier**: supporting | **Domains**: silicon

High-purity vitreous silica (SiO₂) crucibles used in Czochralski crystal growth, manufactured by arc-fusing natural quartz sand into a transparent amorphous vessel. Must withstand 1420°C silicon melt temperature while minimizing oxygen dissolution into the crystal; single-use (one pull per crucible).
