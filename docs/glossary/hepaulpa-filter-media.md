# HEPA/ULPA filter media

> **Type**: noun | **Tier**: supporting | **Domains**: chemistry

Filter media for High-Efficiency Particulate Air (99.97% at 0.3 μm) and Ultra-Low Penetration Air (99.999% at 0.12 μm) filters, constructed from randomly arranged microfine glass fiber mats. These media are essential for semiconductor cleanroom air filtration, trapping particles that would destroy integrated circuits.
