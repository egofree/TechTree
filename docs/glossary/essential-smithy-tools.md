# Essential smithy tools

> **Type**: noun | **Tier**: supporting | **Domains**: metals

Fundamental blacksmithing equipment: forge or hearth, anvil, hammers (cross-peen, ball-peen, sledge), tongs (various jaw shapes), fuller, swage, hardy, punch, chisel, and slack tub. These tools enable all basic forging operations from drawing out to punching and bending.
