# Ash fusion temperature

> **Type**: equipment | **Tier**: important | **Domains**: energy

Critical for all combustion methods — if ash melts (typically 1100-1400°C for bituminous coal), it forms clinker that blocks grates and gasifiers. High-ash-fusion coal (>1400°C) is preferred for stoker firing. Low-ash-fusion coal can be used in slagging gasifiers where molten ash is tapped continuously, like blast furnace slag.
