# Multi-source deposition

> **Type**: equipment | **Tier**: important | **Domains**: optics

For coatings requiring more than one material (all multi-layer designs), the vacuum chamber contains two or more evaporation sources, each in a separate crucible with its own shutter. The operator (or automated controller) opens one shutter, deposits the first layer to target thickness, closes that shutter, then opens the next source's shutter for the second layer.
