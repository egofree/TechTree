# Screening process

> **Type**: noun | **Tier**: important | **Domains**: energy

Roller screens (rotating parallel shafts with discs that pass undersize) or vibrating screens with multi-deck mesh. Screening throughput: 100-500 tonnes/hour per screen unit. Fines generation during handling: each transfer point (conveyor-to-conveyor, conveyor-to-bin) generates ~1-3% additional fines from breakage.
