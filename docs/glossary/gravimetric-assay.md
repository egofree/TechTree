# Gravimetric assay

> **Type**: material | **Tier**: important | **Domains**: mining

Dissolve sample, precipitate the target element as an insoluble compound, filter, dry, and weigh. Barium precipitates as BaSO₄ (barium sulfate). Nickel precipitates as NiDMG (nickel dimethylglyoxime, bright red). Accuracy ±0.1% for major element concentrations.
