# Built-in self-test (BIST)

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

On-chip hardware generates test patterns and checks responses without external test equipment. Memory BIST (MBIST) is nearly universal — SRAM arrays are tested by writing/reading address patterns (march tests, checkerboard, galloping patterns). Logic BIST (LBIST) uses pseudo-random pattern generators (LFSRs) and multiple-input signature registers (MISRs) for random logic testing.
