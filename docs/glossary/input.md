# Input

> **Type**: material | **Tier**: important | **Domains**: computing

Punched cards (adapted from Jacquard loom) — operation cards specify which calculation to perform, variable cards specify which store columns to read/write. Program = sequence of cards. Conditional branching achieved by reversing card chain direction (advance or back up based on result sign). The first general-purpose programmable computer concept, though never completed in Babbage's lifetime.
