# Logic gates

> **Type**: noun | **Tier**: important | **Domains**: computing

Grid biased to cutoff (tube off = logic 0) or conduction (tube on = logic 1). Resistor in plate circuit converts current to voltage. Two tubes with common plate resistor = NOR gate. NOR is functionally complete — any Boolean function implementable with NOR gates alone. Inverting output gives OR. Cascading gives AND.
