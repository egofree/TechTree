# ENIAC-era architecture

> **Type**: noun | **Tier**: important | **Domains**: computing

Heat dissipation required forced-air cooling. - **Architecture**: 20 accumulators (each a 10-digit decimal counter using ring circuits), multiplier, divider/square-root unit, function tables (read-only lookup via switches), master clock (100 kHz ring oscillator).
