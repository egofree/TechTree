# Test cost per die

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Wafer sort (probe test): $0.01-0.10 per die depending on complexity. Package test: $0.05-0.50 per die. Burn-in: $0.10-1.00 per die (depends on duration, 24-168 hours). For a complex SoC (system-on-chip), total test cost can be 5-10% of the die cost. Test cost increases with pin count, test speed, and test time.
