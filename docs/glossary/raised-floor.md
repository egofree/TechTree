# Raised floor

> **Type**: noun | **Tier**: important | **Domains**: foundations, photolithography

Perforated steel or aluminum floor panels (25% open area for return airflow) supported on adjustable steel pedestals. Plenum space below the floor serves as the return air path. Floor height: 300-600 mm above structural slab. Floor panels are anti-static (conductive coating grounded to building earth).
