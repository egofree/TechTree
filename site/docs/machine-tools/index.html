Machine Tools Bootstrap

Precision manufacturing capability built via Gingery-style iterative bootstrap: casting → crude lathe → improved lathe → shaper → mill → drill press → surface grinder. Covers metal forming, machining, joining, and casting. The single most important parallel track — without machine tools, no semiconductor equipment can be built.

Capabilities

  • Bearings, Abrasives & Cutting Tools

    Plain journal bearings (babbitt metal), rolling element ball bearings (52100 steel), natural abrasives (emery, sandstone) and synthetic abrasives (SiC, Al₂O₃ from electric arc furnace), cutting tool materials (carbon steel, HSS), thread cutting on lathe, and tap and die production.

  • Iterative Machine Bootstrap

    The core self-improving sequence: crude lathe (0.5 mm tolerance) → improved lathe with leadscrew (0.1 mm) → shaper (0.05 mm) → milling machine (0.025 mm) → drill press → surface grinder (0.005 mm). Each machine doubles precision achievable with the previous generation.

  • Machining

    Lathe turning, milling, drilling, and grinding of metal parts to precise dimensions.

  • Metal Casting

    Crucible furnaces for aluminum (660°C) and cast iron (1200-1400°C), green sand molding with bentonite binder, pattern making with shrinkage allowance, flask-based casting, and investment casting. Every machine tool starts as a sand-cast iron or aluminum casting.

  • Metal Forming

    Forge work, hammer forming, rolling, and drawing of metals into shapes.

  • Metal Joining

    Forge welding, brazing, soldering, and riveting of metal components. Also covers oxy-acetylene welding and cutting (3100°C flame) and arc welding (SMAW/stick) with flux-coated electrodes.
