# Chromium and nickel

> **Type**: noun | **Tier**: supporting | **Domains**: metals

Key alloying elements in stainless steels: chromium provides corrosion resistance (forms Cr₂O₃ passive layer, minimum 10.5% for stainless classification) while nickel stabilizes the austenitic phase and improves ductility and toughness. Together they define the 300-series stainless steels.
