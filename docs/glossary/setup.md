# Setup

> **Type**: material | **Tier**: critical | **Domains**: computing, health, machine-tools

The preparatory configuration of equipment, workspace, or instruments before a process begins. Proper setup determines the quality and safety of the subsequent operation.

## Context in the Tech Tree

Setup procedures appear across domains. In [Medicine](../health/medicine.md), surgical setup requires sterile instruments (boiled 20+ minutes), hand washing with soap and ethanol, and sterile drapes. In [Mechanical Computing](../computing/mechanical.md), setup time for differential analyzers takes hours to days to reconfigure interconnections. In [Bearings & Abrasives](../machine-tools/bearings-abrasives.md), machine setup involves alignment, lubrication, and calibration before operation.

## Technical Details

Surgical setup: clean room, boil all instruments 20+ minutes for sterilization. Surgeon washes hands and forearms with soap and water, then soaks in 70% ethanol or 2% phenol solution. Sterile drapes cover the patient. Good lighting — daylight or focused oil lamps — is essential. This basic aseptic technique dramatically reduces infection rates from near-certain to manageable.

Machine tool setup: workpiece must be securely clamped, cutting tool aligned to the correct position and angle, and speeds/feeds set appropriately for the material. Incorrect setup causes chatter, poor surface finish, tool breakage, or workpiece damage.

Computing setup: mechanical computers require precise alignment of shafts, gears, and integrator disks. Each integrator must be calibrated to the correct function coefficient. Interconnections between integrators define the differential equation to be solved.
