# npn transistor

> **Type**: material | **Tier**: important | **Domains**: computing

Emitter grounded, base driven high → collector current flows (transistor on, Vce ≈ 0.2 V saturation). Base driven low → collector current ceases (transistor off, Vce ≈ supply voltage). One transistor + one resistor = digital inverter. Switching speed: 1-100 MHz depending on construction (alloy junction = slow, diffused planar = fast).
