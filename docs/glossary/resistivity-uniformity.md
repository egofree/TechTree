# Resistivity uniformity

> **Type**: noun | **Tier**: important | **Domains**: silicon

±5-10% axial, ±2-5% radial - **Oxygen content**: 10-20 ppma (from crucible dissolution) - **Carbon content**: <1 ppma (from heater/atmosphere) - **Dislocation density**: <100/cm² (ideally zero — Dash process achieves this) - **Crystal orientation**: Within ±0.5° of target
