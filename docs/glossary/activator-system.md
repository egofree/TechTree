# Activator system

> **Type**: noun | **Tier**: important | **Domains**: polymers

Zinc oxide (3-5 phr) + stearic acid (1-2 phr). The stearic acid solubilizes ZnO, forming zinc stearate in situ, which then complexes with the accelerator to activate sulfur cross-linking. Without activators, sulfur cure is extremely slow (>8 hours at 150°C).
