# Parchment preservation

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Parchment is hygroscopic and dimensionally unstable. It curls below 40% RH and softens above 60% RH. The gelatin collagen matrix is susceptible to acid hydrolysis, particularly if exposed to atmospheric sulfur dioxide (forms sulfuric acid). Monitor pH of parchment surface periodically; values below pH 4.5 indicate active degradation requiring intervention. Storage: 18-22°C, 45-55% RH, flat storage in acid-free folders, no folding (cracks the collagen fibers).
