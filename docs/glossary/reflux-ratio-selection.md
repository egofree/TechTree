# Reflux ratio selection

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Reflux ratio (R = L/D, where L = reflux flow, D = distillate flow) is the primary operating variable controlling separation sharpness. Minimum reflux ratio (R_min) is the lowest value that achieves the specified separation, determined from the McCabe-Thiele diagram intersection of the rectifying line with the equilibrium curve.
