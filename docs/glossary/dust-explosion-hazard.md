# Dust explosion hazard

> **Type**: noun | **Tier**: important | **Domains**: silicon

Silicon powder finer than 100 μm is a combustible dust. When suspended in air at concentrations above ~100 g/m³, it can ignite and explode. The minimum ignition energy is ~100 mJ (low enough for static discharge). Kst value (dust explosion class): ~100-200 bar·m/s (St class 1, moderate explosion hazard).
