# Beeswax

> **Type**: material | **Tier**: important | **Domains**: animals

(melting point 62-64°C, density 0.95 g/cm³): - **Investment casting (lost-wax method)**: Critical application for metallurgy — see [metals.copper-bronze](../metals/copper-bronze.md). Wax pattern → ceramic shell mold → melt out wax → pour molten metal. Enables complex shapes impossible with sand casting. - **Writing tablets**: Wooden frame filled with beeswax, inscribed with stylus, smoothed by gentle heating.
