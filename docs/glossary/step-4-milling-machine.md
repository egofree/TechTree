# Step 4: Milling machine

> **Type**: material | **Tier**: important | **Domains**: machine-tools

Step 4: Milling machine (tolerance ~0.025 mm) — Function**: Rotating multi-tooth cutter removes material. More versatile than shaper — handles pockets, profiles, helical surfaces (gears). - **Spindle**: Driven by belt or gear from motor/line shaft. Speed 50-2000 RPM (adjustable via pulley changes). - **Table**: X-Y movement via leadscrews with graduated dials (0.025 mm divisions = "one thou"). - **Use it to make**: gear blanks, slots, flat reference surfaces, precision holes via boring head.
