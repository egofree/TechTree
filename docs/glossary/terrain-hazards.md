# Terrain hazards

> **Type**: noun | **Tier**: important | **Domains**: mining

Prospecting involves traversing rugged, unstable terrain. Rockfalls, loose scree, steep slopes. Sturdy boots, careful footing. Never prospect alone in remote areas. Carry a basic first aid kit, map, compass, whistle, and emergency shelter blanket. A personal locator beacon (PLB) provides satellite distress signaling in areas without cell coverage.
