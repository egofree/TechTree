# Lime softening

> **Type**: noun | **Tier**: important | **Domains**: health

Lime softening (permanent + temporary) — Add slaked lime Ca(OH)₂ to hard water. Ca(HCO₃)₂ + Ca(OH)₂ → 2CaCO₃↓ + 2H₂O. Magnesium hardness: Mg(HCO₃)₂ + 2Ca(OH)₂ → Mg(OH)₂↓ + 2CaCO₃↓ + 2H₂O. Settle, filter. Reduces total hardness significantly. Essential for boiler feedwater to prevent scale buildup. See: [Water Supply & Sanitation](../health/sanitation.md)
