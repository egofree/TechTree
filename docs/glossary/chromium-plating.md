# Chromium plating

> **Type**: noun | **Tier**: supporting | **Domains**: metals

Electroplating of a thin (0.025-0.5 μm) chromium layer onto metal surfaces for hardness (900-1100 HV), wear resistance, and corrosion protection. Uses chromic acid (H₂CrO₄) electrolyte; hard chrome plating (2-500 μm) restores worn parts.
