# Proto-Sinaitic script

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Proto-Sinaitic script (~1800 BCE) — Semitic workers in Egyptian turquoise mines adapted Egyptian hieroglyphic symbols to represent Semitic consonantal sounds (acrophonic principle — the word for "house" *bayt* → the symbol for /b/). Approximately 27 consonant signs. No vowel notation. This was the first true alphabet — one symbol per phoneme. See: [Writing & Record-Keeping](../knowledge/writing.md)
