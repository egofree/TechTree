# Cleaning procedure

> **Type**: material | **Tier**: important | **Domains**: gas-handling

Wash all oxygen-contact surfaces in hot alkaline detergent solution. Rinse with deionized water. Blow dry with oil-free nitrogen or clean air. Inspect visually (no visible oil, grease, or particles). Test with UV light (hydrocarbon residues fluoresce). Assemble with clean tools (dedicated oxygen-clean tools, never used on other services).
