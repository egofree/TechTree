# Dope application

> **Type**: material | **Tier**: important | **Domains**: transport

Apply nitrocellulose dope (see below) in multiple thin coats. First coat (clear dope): saturates fabric, bonds to structure. Subsequent coats: build film thickness, fill weave. Aluminum-pigmented dope (silver): UV barrier — prevents sunlight from degrading fabric. Final coat: colored dope for visibility and identification. Total: 6-10 coats, ~100 g/m² dope applied.
