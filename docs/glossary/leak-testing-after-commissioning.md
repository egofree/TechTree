# Leak testing after commissioning

> **Type**: material | **Tier**: important | **Domains**: gas-handling

After pressure testing and before introducing process gas, perform a sensitive leak test on the completed system. For flammable gas systems, pressurize with nitrogen at operating pressure and check all joints with soap solution (bubble test). For higher sensitivity, pressurize with a helium-nitrogen mixture and use a helium sniffer probe to check each joint (sensitivity ~10⁻⁶ atm·cc/s). Document all leak test results.
