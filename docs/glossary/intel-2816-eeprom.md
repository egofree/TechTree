# Intel 2816 EEPROM

> **Type**: equipment | **Tier**: important | **Domains**: computing

Intel 2816 EEPROM (1978) — 16 Kbit (2K × 8) electrically erasable PROM. Byte-level erasure using on-chip charge pumps, no UV source needed. The ancestor of modern Flash memory. Write endurance ~10⁴ cycles for early devices. See: [Electronic Computing](../computing/electronic.md)
