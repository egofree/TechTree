# Harvard Mark I operating detail

> **Type**: noun | **Tier**: supporting | **Domains**: computing

Technical specifics of the IBM Automatic Sequence Controlled Calculator (Harvard Mark I, 1944): 51 feet long, 8 feet tall, electromechanical relays performing addition in 0.3 seconds and multiplication in 6 seconds, programmed via 24-channel paper tape. Demonstrated that automatic computation was practical.
