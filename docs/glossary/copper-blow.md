# Copper blow

> **Type**: process | **Tier**: important | **Domains**: metals

Remaining Cu₂S is oxidized: 2Cu₂S + 3O₂ → 2Cu₂O + 2SO₂, then Cu₂S + 2Cu₂O → 6Cu + SO₂. The product is **blister copper** (98-99% Cu, named for SO₂ blisters trapped during solidification). Duration: 1-3 hours. Temperature: 1150-1250°C.
