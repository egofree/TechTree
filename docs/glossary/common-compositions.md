# Common compositions

> **Type**: noun | **Tier**: supporting | **Domains**: metals

Standard copper-bronze alloy formulations: pure copper (C11000, 99.9% Cu for electrical conductors), cartridge brass (70Cu/30Zn), phosphor bronze (95Cu/5Sn + P), aluminum bronze (90Cu/10Al for marine use), and beryllium copper (98Cu/2Be for springs/tools).
