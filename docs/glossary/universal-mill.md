# Universal mill

> **Type**: noun | **Tier**: important | **Domains**: metals

(for H-beams and wide-flange sections): - Four rolls: two horizontal (reduce web thickness) and two vertical (reduce flange thickness). The rolls act simultaneously, allowing independent control of web and flange dimensions. - H-beam sizes: depth 100-1100 mm, flange width 50-450 mm, web thickness 4-20 mm. Lengths 6-24 m standard. - Tolerances: depth ±2-4 mm, flange width ±2-3 mm, web thickness ±0.5-1.0 mm. Out-of-square on flanges ≤ 1% of flange width.
