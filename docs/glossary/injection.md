# Injection

> **Type**: noun | **Tier**: important | **Domains**: health

Injection (advanced — requires glass syringes + needles) — Intravenous (immediate onset), intramuscular (sustained release). Sterile technique mandatory. Hypodermic needle: drawn-glass syringe + steel needle (requires precision metalworking). See: [Pharmacology](../health/pharmacology.md)
