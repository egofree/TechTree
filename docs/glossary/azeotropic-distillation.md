# Azeotropic distillation

> **Type**: noun | **Tier**: supporting | **Domains**: chemistry

Distillation technique for separating mixtures that form azeotropes (constant-boiling compositions) by adding an entrainer that creates a new, separable azeotrope. Used to break ethanol-water (95.6% azeotrope) with benzene or cyclohexane to achieve anhydrous ethanol.
