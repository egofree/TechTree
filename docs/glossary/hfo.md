# HfO₂

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

HfO₂ (high-k gate dielectric) — Tetrakis(dimethylamido)hafnium (TDMAH) + H₂O. Temperature: 200-300°C. Growth: ~1.0 Å/cycle. Replaces SiO₂ gate oxide at sub-45 nm nodes. See: [Advanced Processes](../vlsi-scaling/advanced-processes.md)
