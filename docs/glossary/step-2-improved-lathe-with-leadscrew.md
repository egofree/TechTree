# Step 2: Improved lathe with leadscrew

> **Type**: noun | **Tier**: important | **Domains**: machine-tools

Step 2: Improved lathe with leadscrew (tolerance ~0.1 mm) — Leadscrew**: First leadscrew is hand-cut (file or chisel into round bar, using thread gauge for checking). Not precise, but functional. - **Half-nut mechanism**: Split nut that engages/disengages leadscrew for threading. - **Thread cutting**: Mount change gears between spindle and leadscrew. Gear ratio determines thread pitch. Start with standard metric or imperial pitches. - Use it to make: leadscrew for shaper, accurate threaded fasteners, taps and dies.
