# Regulation

> **Type**: noun | **Tier**: important | **Domains**: computing

Series pass transistor (or IC regulator like 7805 for +5V, 7812 for +12V) maintains constant output voltage despite input variation and load changes. Dropout voltage: 2-3V (input must be at least 2-3V above output). Efficiency: Output power / Input power ≈ Vout/Vin (typically 30-60% — the rest is dissipated as heat in the pass transistor). Heat sinking required.
