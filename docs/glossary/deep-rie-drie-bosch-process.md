# Deep RIE (DRIE) — Bosch Process

> **Type**: noun | **Tier**: supporting | **Domains**: vlsi-scaling

The Bosch process adapted for advanced semiconductor nodes, using time-multiplexed etch-passivate cycles to create deep, high-aspect-ratio features in silicon. Critical for forming through-silicon vias (TSVs), MEMS structures, and 3D integration interconnects.
