# Wet scrubbers

> **Type**: material | **Tier**: important | **Domains**: chemistry

Packed-bed scrubbers using NaOH solution (5-15%) remove acid gases (HF, HCl, Cl₂, SiF₄) from exhaust streams. Gas flows upward through packing material while scrubbing solution flows downward. Mass transfer: acid gas + NaOH → sodium salt + water. For fluorine-containing gases: HF + NaOH → NaF + H₂O. Scrubber solution monitored for pH (maintain >10 for efficient absorption) and fluoride concentration. Spent scrubber solution treated for fluoride removal before discharge.
