# Bias circuits

> **Type**: noun | **Tier**: important | **Domains**: silicon

Operating point (quiescent point) set by base bias resistors. Fixed bias (single base resistor, β-dependent). Voltage divider bias (two resistors from Vcc to ground with base at junction, most stable against β variation). Emitter resistor provides negative feedback: if Ic increases → Ie increases → Ve increases → Vbe decreases → Ic decreases. Self-stabilizing.
