# Lattice method for multi-digit multiplication

> **Type**: noun | **Tier**: important | **Domains**: computing

To multiply 467 × 38, arrange rods 4, 6, 7 side by side. For the 8 row: read 3/2, 4/8, 5/6 across the three rods. Sum each diagonal column right to left with carries: 6, 2+8+5=15 (write 5, carry 1), 3+4+1=8, 3 = 3736. Repeat for the 3 row (rod arrangement ×3 = 1401), shifted one position left. Sum the partial products: 3736 + 14010 = 17746.
