# SQ 2: Measurement, Timekeeping &amp; Metrology

**Starts**: Phase 1–3  
**Priority**: Critical — you cannot build what you cannot measure.

## Problem

Precision manufacturing (Phase 3) and all semiconductor work requires consistent, calibrated measurement. Without standardized units and measuring instruments, interchangeable parts and repeatable processes are impossible.

## Technologies &amp; Systems

### Base Unit Standards
- **Length**: Define reference standard (e.g., a particular metal bar at defined temperature)
- **Mass**: Reference weight standards
- **Volume**: Calibrated containers
- **Temperature**: Reference points (water freezing/boiling) → temperature scales

### Timekeeping
- **Water clocks / sand glasses**: Early time measurement
- **Mechanical clocks**: Escapement mechanisms → pendulum clocks → chronometers
  - Requires precision gear-cutting (from Phase 3 machine tools)
  - Critical for: process timing, astronomical observation, navigation
- **Electrical clocks** (later): Synchronous motor clocks tied to AC line frequency

### Precision Instruments
- Rules and straight edges (calibrated against master standards)
- Calipers (vernier, later dial/digital)
- Micrometers (iterative improvement of screw threads)
- Gauge blocks (Jo-blocks — stacked for precise lengths)
- Go/no-go gauges for production quality control
- Angle measurement (protractors, sine bars)

### Calibration Infrastructure
- Temperature-controlled calibration rooms
- Master reference standards (maintained and periodically verified)
- Calibration traceability chain
- Inter-laboratory comparisons (as scale grows)

### Electrical Standards
- **Voltage**: Standard cells (Weston cadmium cell ≈ 1.0186V)
- **Resistance**: Standard resistors (manganin wire)
- **Current**: Derived from voltage + resistance (Ohm's law)
- **Frequency**: Crystal oscillators (later, once quartz crystals available)

## Integration Points

| Phase | Contribution |
|-------|-------------|
| Phase 1 | Basic length, volume, weight standards |
| Phase 3 | Precision instruments, gauge blocks, thread standards |
| Phase 4 | Electrical measurement standards, voltmeters, ammeters |
| Phase 6 | Optical metrology, interferometry |
| Phase 7 | Crystal orientation measurement, wafer thickness/flatness |
| Phase 8 | Feature size measurement, overlay registration, defect inspection |

## Key Deliverables

- Defined base unit system with physical reference standards
- Accurate clocks (mechanical → electrical)
- Calibrated precision instruments (calipers, micrometers, gauges)
- Calibration laboratory and traceability system
