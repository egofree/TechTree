# Particle count

> **Type**: noun | **Tier**: important | **Domains**: chemistry

ISO 4406 cleanliness code — critical for hydraulic systems (target: 16/14/11 for servo valves). (4) **Spectrometric analysis**: ICP or atomic absorption detects wear metals (Fe, Cu, Pb, Sn, Cr) at ppm levels — trending indicates which component is wearing. (5) **Water content**: Karl Fischer titration — water causes rust, reduces film strength, promotes microbial growth. Limits: <0.1% for most systems, <0.03% for turbine oils.
