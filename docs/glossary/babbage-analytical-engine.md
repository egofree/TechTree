# Babbage Analytical Engine

> **Type**: material | **Tier**: important | **Domains**: computing

Babbage Analytical Engine (conceptual, requires advanced Machine Tools) — - **Store**: Columns of toothed wheels representing variable values (1000 columns × 50 digits each proposed). Memory for data and intermediate results — the conceptual ancestor of RAM. - **Mill**: Set of gears and carries that perform arithmetic operations (add, subtract, multiply, divide) on values fetched from the store.
