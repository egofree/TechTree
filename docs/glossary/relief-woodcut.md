# Relief woodcut

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Same technique as woodblock text, but carving images rather than letters. The illustrator draws on the block; the carver removes background. Woodcut illustrations can be printed alongside text on the same page by cutting the image block to type-height and locking it in the forme with the type. The earliest printed books with illustrations (e.g., the Nuremberg Chronicle, 1493) used this technique. Line work only — tone is suggested by hatching (parallel lines) and cross-hatching.
