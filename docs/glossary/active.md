# Active

> **Type**: noun | **Tier**: important | **Domains**: silicon

Active (base-emitter forward biased, base-collector reverse biased: Ic = β·Ib, amplifier operation). **Saturation** (both junctions forward biased: Vce ≈ 0.2V, switch "on" state). **Reverse active** (junctions reversed, β very low, rarely used). See: [Basic Semiconductor Devices](../silicon/basic-devices.md)
