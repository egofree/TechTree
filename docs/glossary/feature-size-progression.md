# Feature size progression

> **Type**: noun | **Tier**: supporting | **Domains**: vlsi-scaling

Historical reduction in minimum transistor feature dimensions from 10 μm (1970s) through 1 μm (1980s), 100 nm (2000s), to sub-10 nm (2020s), driven by photolithography wavelength reduction and resolution enhancement techniques. Each generation doubles transistor density per Moore's Law.
