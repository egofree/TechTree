# Gas line layout

> **Type**: equipment | **Tier**: important | **Domains**: chemistry

Each gas follows the path: cylinder → gas cabinet (with automatic shutoff valve) → stainless steel trunk line → gas panel inlet valve → particle filter (0.003 μm sintered metal) → mass flow controller → check valve (prevent backflow) → process chamber. All lines are heat-traced (electric heating cable, 60-80°C) for gases that condense at room temperature (TEOS bp 168°C, TMA boils at -3°C but can decompose to solids in cold lines).
