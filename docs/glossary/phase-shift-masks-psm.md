# Phase Shift Masks (PSM)

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Alternating aperture PSM etches adjacent mask apertures to different depths, creating a 180° phase shift between them. Destructive interference at feature edges sharpens the aerial image, improving contrast and resolution. Attenuated PSM (partially transmitting chrome) provides a milder benefit with simpler mask fabrication.
