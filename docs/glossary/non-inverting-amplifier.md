# Non-inverting amplifier

> **Type**: noun | **Tier**: important | **Domains**: silicon

Output = (1 + Rf/Rin) × Vin. Input impedance >1 MΩ (depends on op-amp input stage, >10¹² Ω for FET-input op-amps). No phase inversion. Minimum gain = 1 (voltage follower: Rf = 0, Rin = ∞). The voltage follower buffers high-impedance sources without loading them.
