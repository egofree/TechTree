# Depletion width

> **Type**: noun | **Tier**: important | **Domains**: silicon

W = √(2εᵣε₀(Vbi - V)/(qN)), where εᵣ = 11.7 for silicon, ε₀ = 8.854×10⁻¹² F/m, q = 1.6×10⁻¹⁹ C, and N is the dopant concentration on the lightly-doped side. For a typical junction with N = 10¹⁶/cm³ and zero bias: W ≈ 0.3 μm. Under reverse bias, W increases as √V.
