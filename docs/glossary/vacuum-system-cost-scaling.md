# Vacuum system cost scaling

> **Type**: equipment | **Tier**: important | **Domains**: gas-handling

The cost of a vacuum system scales roughly with the chamber volume raised to the 0.7 power and with the inverse of the target pressure. A system for 10⁻⁶ Pa costs ~10× more than one for 10⁻³ Pa of the same volume, primarily due to the need for higher-performance pumps, metal-sealed flanges, and bake-out capability.
