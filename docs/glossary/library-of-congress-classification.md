# Library of Congress Classification

> **Type**: material | **Tier**: important | **Domains**: knowledge

(developed 1897-1901): - Uses alphanumeric notation: single or double letters for main classes (T = Technology, TA = Engineering, TF = Railroad engineering), followed by numbers for subdivision, then a Cutter number (derived from author name) for shelf order. Example: TA405 .C35 = Structural engineering, author with surname starting "Ca". More expressive than Dewey for large, specialized collections — accommodates millions of unique shelf positions.
