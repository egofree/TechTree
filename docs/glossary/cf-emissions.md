# CF₄ emissions

> **Type**: noun | **Tier**: important | **Domains**: metals

Carbon tetrafluoride has a global warming potential (GWP-100) of 6,620 — a single anode effect event can release kg of CF₄. Modern smelters target <0.1 anode effects per pot-day (some achieve <0.01) through automated feed control and rapid alumina feeding on voltage rise detection.
