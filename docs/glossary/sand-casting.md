# Sand casting

> **Type**: process | **Tier**: important | **Domains**: metals

(the primary molding method for cast iron): 1. Make a wooden pattern (slightly oversized to account for shrinkage — ~1% for cast iron). 2. Pack molding sand (silica sand + 5-10% clay binder + 3-5% water) around the pattern in a two-part flask (cope and drag). 3. Separate flask halves, remove pattern → cavity remains. 4. Create sprue (pouring channel) and risers (reservoirs to feed shrinkage). Add cores (baked sand shapes) for internal cavities if needed. 5. Close flask, clamp halves together....
