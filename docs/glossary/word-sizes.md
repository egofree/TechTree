# Word sizes

> **Type**: noun | **Tier**: important | **Domains**: computing

Define data width and address space. 8-bit (microcontrollers, early hobby computers), 12-bit (PDP-8), 16-bit (PDP-11, early minicomputers), 32-bit (IBM System/360, VAX). Larger word = larger directly-addressable memory (16-bit → 64 KB, 32-bit → 4 GB) and wider arithmetic. Word size determines ALU width, register width, data bus width, and memory alignment.
