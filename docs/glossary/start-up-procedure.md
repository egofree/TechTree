# Start-up procedure

> **Type**: noun | **Tier**: important | **Domains**: chemistry

From ambient temperature to full production takes 24-72 hours. Phase 1 (0-8 hours): start main compressor, establish air flow through molecular sieve adsorbers and heat exchangers. The cold box warms initially as ambient-temperature air flows through it. Phase 2 (8-24 hours): expansion turbine begins cooling.
