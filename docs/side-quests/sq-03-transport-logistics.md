# SQ 3: Transportation, Logistics &amp; Communication

**Starts**: Phase 2–4  
**Priority**: High — industrial-scale operations require moving materials, people, and information.

## Problem

Semiconductor fabrication requires raw materials from diverse sources (quartz, iron, coal, fluorite, copper, sulfur, etc.), concentrated labor forces, and coordination across distances. Without transportation and communication infrastructure, operations remain village-scale.

## Technologies &amp; Systems

### Transportation
- **Roads and bridges**: Graded, drained road surfaces; stone or timber bridges
- **Wheeled carts and wagons**: Standardized axle widths and wheel designs
- **Water transport**: Canals, barges, ports (for bulk materials — far cheaper than land)
- **Railways** (Phase 4+): Iron rails + steam locomotives for high-volume transport
  - Standard gauge, standardized containers
- **Later**: Internal combustion engines, motor vehicles

### Logistics
- **Standardized containers**: Uniform box sizes for handling efficiency
- **Inventory management**: Ledger systems, later computerized tracking
- **Supply chain coordination**: Scheduled deliveries, buffer stocks, multi-site coordination
- **Warehousing**: Dry, organized storage for raw materials and finished goods

### Communication
- **Writing and couriers** (Phase 1): Basic message transmission
- **Printing press** (Phase 2–3): Mass information dissemination
- **Semaphore / signal towers**: Long-distance visual communication
- **Electric telegraph** (Phase 4): Near-instantaneous long-distance communication
  - Copper wire (from Phase 3 wire drawing) + batteries/generators
  - Morse code or similar binary encoding
  - Enables: coordination of distant operations, rapid error reporting
- **Telephone** (later): Voice communication over wire

## Integration Points

| Phase | Contribution |
|-------|-------------|
| Phase 1 | Paths, basic transport, writing |
| Phase 2 | Iron tools for road building, wheeled vehicles |
| Phase 3 | Precision parts for rail, standardization |
| Phase 4 | Steam transport, telegraph, railways |
| Phase 7–8 | Specialized transport for hazardous chemicals, cleanroom supplies |

## Key Deliverables

- Road network connecting mining, processing, and manufacturing sites
- Standardized transport vehicles and containers
- Working telegraph communication network
- Inventory and logistics management system
