# ISO 12944 corrosivity categories

> **Type**: material | **Tier**: important | **Domains**: chemistry

C1 (very low, heated interiors), C2 (low, rural atmospheres), C3 (moderate, urban/coastal), C4 (high, industrial/coastal), C5 (very high, aggressive industrial/marine), CX (extreme, offshore/splash zone). Each category specifies minimum DFT, number of coats, and expected time to first maintenance.
