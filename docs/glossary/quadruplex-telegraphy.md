# Quadruplex telegraphy

> **Type**: material | **Tier**: important | **Domains**: transport

Sending four messages on a single wire (two in each direction). Combines duplex operation with a polarized relay that responds to current direction (positive/negative) as well as current on/off. One message uses current polarity, the other uses current on/off, and both are independent. Invented by Edison in 1874. Quadruplex increased line revenue 4× without new construction, making long-distance telegraphy far more profitable.
