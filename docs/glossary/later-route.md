# Later route

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Later route (Ostwald process, requires ammonia from Haber-Bosch) — NH₃ + air over Pt-Rh catalyst at 850°C → NO → NO₂ → H₂O absorption → HNO₃. Much larger scale but requires ammonia synthesis via the Haber-Bosch process. See: [Mineral Acid Production](../chemistry/acids.md)
