# Key reliability tests

> **Type**: noun | **Tier**: important | **Domains**: chemistry

(1) **Temperature cycling**: -65°C to +150°C, 500-1000 cycles. Detects: die attach voids, wire bond degradation, solder joint fatigue, package cracking. (2) **High temperature operating life (HTOL)**: 125°C at nominal voltage, 1000 hours. Detects: gate oxide breakdown, electromigration, hot carrier degradation.
