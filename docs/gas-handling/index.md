# Gas Handling

Capabilities in this domain:

- [Basic Gas Handling](basic.md) — Gas compression, storage, and purification infrastructure including piping, valves, and pumps for corrosive service.
- [Vacuum Technology](vacuum.md) — Piston pumps (rough vacuum 10-50 Torr), rotary vane pumps (workhorse, 10⁻²-10⁻³ Torr), diffusion pumps (high vacuum 10⁻⁶-10⁻⁸ Torr), vacuum chambers (steel or glass), vacuum measurement (Bourdon, Pirani, ion gauges), and leak detection (bubble test, helium leak detector).
