# Solvay process

> **Type**: equipment | **Tier**: important | **Domains**: chemistry

Solvay process (more efficient, later) — Principle**: NaCl + NH₃ + CO₂ + H₂O → NaHCO₃ (precipitates) + NH₄Cl. Heat NaHCO₃ → Na₂CO₃ + CO₂ + H₂O. Recycle NH₃ and CO₂. - **Ammonia recovery**: Heat NH₄Cl with Ca(OH)₂ (slaked lime) → NH₃ + CaCl₂ + H₂O. Calcium chloride is waste product. - **CO₂ source**: Heat limestone (CaCO₃ → CaO + CO₂) in lime kiln.
