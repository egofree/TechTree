# Electrotype

> **Type**: material | **Tier**: important | **Domains**: knowledge

(electroplating, ~1840): Grow a thin copper shell on a wax impression of the type surface by electroplating (copper deposited from copper sulfate solution using electric current). Back the copper shell with type metal for rigidity. Produces a more faithful reproduction than stereotype casting (no shrinkage or distortion from molten metal). Requires electrical capability.
