# Oil viscosity grades

> **Type**: material | **Tier**: important | **Domains**: chemistry

(ISO VG system): ISO VG 32 = kinematic viscosity 32 cSt at 40°C (light spindle oil). ISO VG 68 = 68 cSt (general machine oil). ISO VG 220 = 220 cSt (gear oil). ISO VG 460 = 460 cSt (heavy gear oil). Higher viscosity = thicker oil = more load capacity but more viscous friction (heat).
