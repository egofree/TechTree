# CaO-SiO₂

> **Type**: noun | **Tier**: important | **Domains**: silicon

Most common. Weight ratio CaO:SiO₂ = 40:60 to 60:40. Basicity (CaO/SiO₂) ~0.7-1.5. Additives of CaF₂ (5-15 wt%) reduce slag viscosity and improve kinetics. This system requires only lime and silica — both available early in the bootstrap chain.
