# Hand press

> **Type**: material | **Tier**: important | **Domains**: knowledge

(Gutenberg-style common press, 1440-1800): 250 sheets/hour single-color, requiring a skilled two-operator team (beater inks the type, puller operates the press). Each sheet requires one pull of the bar. A full working day (10-12 hours) produces 2,000-3,000 impressions, but only half are usable on each side of a sheet, so effective production for a two-sided sheet is ~1,250 sheets/day. Setup time (make-ready) adds 1-2 hours per forme before production begins.
