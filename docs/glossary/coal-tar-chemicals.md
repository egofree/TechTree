# Coal tar chemicals

> **Type**: material | **Tier**: important | **Domains**: polymers

Coal carbonization (coke ovens, 1000-1200°C) produces coal tar containing phenol, cresols, naphthalene, benzene, toluene, and xylene. Historically the primary source of phenol (for Bakelite), benzene (for nylon), and other aromatic monomers before petroleum-derived routes became dominant.
