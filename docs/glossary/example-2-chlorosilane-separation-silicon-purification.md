# Example 2: Chlorosilane separation (silicon purification)

> **Type**: noun | **Tier**: supporting | **Domains**: chemistry

Distillation of chlorosilane mixtures (SiHCl₃, SiCl₄, SiH₂Cl₂) for silicon purification, exploiting different boiling points (SiHCl₃: 31.8°C, SiCl₄: 57.6°C) to achieve electronic-grade purity. Multiple distillation columns in series achieve the required 9-11N purity.
