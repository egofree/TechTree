# Host trees

> **Type**: material | **Tier**: important | **Domains**: polymers

The lac bug feeds on a range of host trees. Primary hosts in India: *Schleichera oleosa* (kusum), *Butea monosperma* (palas/dhak), *Ziziphus mauritiana* (ber). Secondary hosts: *Acacia catechu* (khair), *Ficus religiosa* (peepal), *Gmelina arborea* (gamhar), and several other species.
