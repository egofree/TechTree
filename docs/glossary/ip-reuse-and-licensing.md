# IP reuse and licensing

> **Type**: material | **Tier**: important | **Domains**: vlsi-scaling

Standard cell libraries, memory compilers, I/O pad cells, and processor cores (soft IP in HDL, hard IP as placed layout) are licensed building blocks. A modern SoC may integrate 50-200 IP blocks from multiple vendors. IP quality directly impacts yield — a faulty IP block can kill an entire chip design. Verification of third-party IP is a significant engineering challenge.
