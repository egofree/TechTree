# Thermocouple use

> **Type**: noun | **Tier**: important | **Domains**: metals

Type J (iron-constantan, 0-760°C), Type T (copper-constantan, -200 to 370°C), and Type E (chromel-constantan, -200 to 900°C) thermocouples all use constantan as one leg. Seebeck coefficient: ~50-60 μV/°C for Cu-Constantan pair. Calibration tables standardized (NIST ITS-90). Thermocouple wire is drawn to 0.1-1.0 mm diameter, insulated with fiberglass or ceramic fiber. Junction formed by twisting and welding (gas or resistance weld).
