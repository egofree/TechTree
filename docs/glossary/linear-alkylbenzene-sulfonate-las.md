# Linear alkylbenzene sulfonate (LAS)

> **Type**: noun | **Tier**: important | **Domains**: chemistry

— the workhorse synthetic detergent: - **Feedstock**: Linear alkenes (C10-C14) from petroleum cracking (see [Petroleum](petroleum-alternatives.md)) or linear paraffins from Fischer-Tropsch synthesis. Benzene from coal tar or petroleum refining. - **Alkylation**: React linear alkene with benzene using HF or AlCl₃ catalyst → linear alkylbenzene (LAB). - **Sulfonation**: React LAB with SO₃ (or concentrated H₂SO₄, or oleum) → linear alkylbenzene sulfonic acid (LABSA).
