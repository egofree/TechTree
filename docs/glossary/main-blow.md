# Main blow

> **Type**: noun | **Tier**: important | **Domains**: metals

Main blow (minutes 5-15) — C + ½O₂ → CO (with some secondary combustion CO + ½O₂ → CO₂ above the bath). Carbon drops from ~4% to the target 0.03-0.10%. Massive CO generation — BOF off-gas is 70-85% CO with a calorific value of ~8 MJ/Nm³, collected and used as fuel gas in the steelworks. See: [Modern Steelmaking](../metals/steelmaking.md)
