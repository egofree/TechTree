# Map and drawing storage

> **Type**: material | **Tier**: important | **Domains**: knowledge

Flat files (plan chests) — wide, shallow drawers (A0 size: 850 mm × 1200 mm, 50 mm deep) for engineering drawings, maps, and architectural plans. Store flat, never rolled (rolling causes cracking and makes retrieval difficult). Label each drawer with content range. Acid-free interleaving tissue between items.
