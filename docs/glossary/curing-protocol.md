# Curing protocol

> **Type**: noun | **Tier**: supporting | **Domains**: chemistry

Time-temperature regime for hydration of Portland cement, typically 20°C minimum for 28 days to achieve design strength, with initial set within 45 minutes and final set within 8 hours. Accelerated curing at 60-80°C using steam reduces curing time to 1-3 days for precast products.
