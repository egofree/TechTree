# Valve types

> **Type**: noun | **Tier**: important | **Domains**: gas-handling

CGA (Compressed Gas Association) standard connections — each gas type has a unique thread/pin pattern to prevent cross-connection (e.g., CGA 580 for inert gases, CGA 350 for flammable gases, CGA 540 for oxygen). Valve outlet threads are right-hand for non-fuel gases and left-hand for fuel gases as an additional safety check.
