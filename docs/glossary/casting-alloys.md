# Casting alloys

> **Type**: material | **Tier**: important | **Domains**: metals

(e.g., A356 = Al-7Si-0.3Mg): Silicon improves castability (fluidity, reduced shrinkage). Heat-treatable via Mg₂Si precipitation. Used for engine blocks, transmission cases, wheels. Al-Si eutectic (12.6% Si) provides the best fluidity — hypereutectic alloys (e.g., A390, 17% Si) used for wear-resistant cylinder liners where primary Si particles provide abrasion resistance.
