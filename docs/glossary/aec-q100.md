# AEC-Q100

> **Type**: material | **Tier**: important | **Domains**: chemistry

for automotive electronics: more stringent than JEDEC — temperature range -40 to +150°C (Grade 0), zero defects philosophy, additional tests (electromigration, soft error rate). **MIL-STD-883** for military/space: even more stringent — radiation testing (total ionizing dose, single event effects), outgassing testing (ASTM E595, <1% total mass loss, <0.1% collected volatile condensable materials for spacecraft materials).
