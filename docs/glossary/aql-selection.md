# AQL selection

> **Type**: material | **Tier**: important | **Domains**: optics

AQL 0.65% for critical optics (laser lenses, photolithography elements, where any defect causes system failure). AQL 1.0% for imaging optics (camera lenses, telescope objectives). AQL 2.5% for general visual optics (viewfinders, illumination lenses, magnifiers). AQL 4.0% for non-critical components (windows, protective covers).
