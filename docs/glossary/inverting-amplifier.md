# Inverting amplifier

> **Type**: noun | **Tier**: important | **Domains**: silicon

Output = -(Rf/Rin) × Vin. Input impedance = Rin. Signal is inverted (180° phase shift). Gain accuracy depends only on resistor ratio, not the op-amp's internal gain. Example: Rf = 100 kΩ, Rin = 10 kΩ → gain = -10.
