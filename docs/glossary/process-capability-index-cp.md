# Process capability index (Cp)

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Ratio of specification tolerance to process spread. Cp = (USL - LSL) / 6σ, where USL = upper spec limit, LSL = lower spec limit, σ = process standard deviation. Cp > 1.33 (process fits within specs with margin) is the minimum for capable processes. Cp = 1.0 means the process just fits specs (0.27% out-of-spec expected). Cp < 1.0 means the process is wider than specs, producing unacceptable defect rates.
