# ULPA filtration

> **Type**: material | **Tier**: important | **Domains**: photolithography

ULPA filtration refers to 99.999% efficient at 0.12 μm for Class 1-10 (ISO 3-4) cleanrooms. Required for sub-micron semiconductor fabrication. Higher pressure drop than HEPA (~2×), requiring more fan power. See: [Cleanrooms](../photolithography/cleanrooms.md)
