# Student workstations

> **Type**: material | **Tier**: important | **Domains**: knowledge

Each student has an individual workstation with the tools and materials for the current exercise. Standardize workstation layout across all positions so the instructor can monitor any student's work at a glance. Minimum workspace: 1.5 m × 0.8 m bench surface per student for most trades; more for large-scale work (foundry, forging).
