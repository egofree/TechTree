# Independent milling machine

> **Type**: noun | **Tier**: important | **Domains**: machine-tools

Cast iron column and base. Knee-type design: the work table rises and falls on a vertical column (knee). Table moves in X and Y via leadscrews with 0.025 mm graduated dials. Spindle driven by belt or gear train, 50-2000 RPM range. The spindle housing swivels for angle milling. First milling machine castings are made in the foundry, then their ways are scraped flat using the surface plate and straightedge from Level 1.
