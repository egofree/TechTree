# Type casting

> **Type**: noun | **Tier**: important | **Domains**: knowledge

The punch (a steel rod with the letter engraved in relief at one end) is struck into a copper blank to create a matrix (negative impression of the letter). The matrix is clamped in a hand mold (type mold) — an adjustable brass or steel frame with a spring release. Molten type metal is poured into the mold with a casting ladle. Cools in seconds.
