# Involute profile

> **Type**: noun | **Tier**: important | **Domains**: computing

Calculator gears use involute tooth profiles (the same geometry as modern machinery gears) for smooth, constant-velocity-ratio power transmission. Generate the profile using a gear hobbing machine (a hobbing cutter shaped like a worm gear, rotating in synchronized mesh with the blank) or a gear shaper (a cutting tool that reciprocates axially while the blank rotates incrementally).
