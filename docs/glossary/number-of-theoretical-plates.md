# Number of theoretical plates

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Determined by McCabe-Thiele method for binary systems. Plot the vapor-liquid equilibrium (VLE) curve (y vs x), draw the 45° diagonal, construct operating lines for rectifying and stripping sections, step off stages between the curve and operating lines. For multicomponent mixtures, the Fenske-Underwood-Gilliland (FUG) method provides short-cut estimates.
