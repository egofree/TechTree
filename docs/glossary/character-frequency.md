# Character frequency

> **Type**: material | **Tier**: important | **Domains**: knowledge

A printer needs far more of some letters than others. A typical type case holds: ~1000 'e', ~800 't', 'a', 'o', ~600 'i', 'n', 's', ~400 'h', 'r', ~200-300 each of less common letters, and ~50 each of rare letters (q, x, z). A full type set for one font size requires 10,000-30,000 individual type pieces, weighing 50-100 kg of type metal.
