# Diisocyanates

> **Type**: noun | **Tier**: important | **Domains**: polymers

Toluene diisocyanate (TDI) — toluene → dinitrotoluene (HNO₃) → reduction to TDA (catalytic hydrogenation) → phosgenation (COCl₂, from CO + Cl₂) → TDI. Or MDI — aniline + formaldehyde → MDA → phosgenation → MDI. Phosgene is extremely toxic (1 ppm lethal) — rigorous containment mandatory.
