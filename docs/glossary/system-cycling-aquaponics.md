# System cycling (aquaponics)

> **Type**: process | **Tier**: critical | **Domains**: agriculture

The 4-6 week process of establishing sufficient nitrifying bacteria colonies before adding fish to an aquaponic system. Add ammonia source, test daily. Ammonia rises then falls, nitrite rises then falls, nitrate appears. System ready when ammonia and nitrite remain at zero within 24 hours of adding ammonia.
