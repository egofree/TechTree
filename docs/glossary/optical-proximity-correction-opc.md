# Optical Proximity Correction (OPC)

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Pre-distorts mask shapes to compensate for diffraction-induced pattern distortions. Lines that are too close together print wider than isolated lines (proximity effect) — OPC adds serifs, hammerheads, and sub-resolution assist features to equalize printed linewidths across all pattern densities. Modern OPC is computed by full-chip simulation on large compute clusters.
