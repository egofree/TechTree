# Sources of CD variation

> **Type**: noun | **Tier**: important | **Domains**: photolithography

Exposure dose non-uniformity (±2% across field), focus variation (±0.2 μm wafer flatness, ±0.1 μm stage leveling), resist thickness variation (±2 nm), development time variation (±2 seconds), post-exposure bake temperature (±1°C), reticle CD error (±20 nm). All contribute to total CD budget.
