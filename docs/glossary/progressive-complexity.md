# Progressive complexity

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Begin with tasks that have a single critical parameter (e.g., filing a flat surface — the parameter is flatness, checked with a straightedge). Progress to tasks with multiple interacting parameters (e.g., turning a shaft — diameter, length, surface finish, and concentricity must all be controlled simultaneously). This builds skill incrementally without overwhelming the learner.
