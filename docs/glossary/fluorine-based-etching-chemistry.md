# Fluorine-based etching chemistry

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Silicon etching relies on generation of atomic fluorine radicals in plasma: CF₄ + e⁻ → CF₃· + F· + e⁻. Fluorine radicals react with silicon: Si + 4F· → SiF₄ (volatile, bp -86°C). Etch rate depends on F· concentration, ion energy, and substrate temperature.
