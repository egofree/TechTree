# Trace elements

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Iron (FeSO₄, 1-50 mg/L), magnesium (MgSO₄, 0.1-1.0 g/L), zinc (ZnSO₄, 0.1-5 mg/L), manganese (MnSO₄, 0.01-1 mg/L), copper (CuSO₄, 0.01-0.5 mg/L). These are enzyme cofactors essential for metabolic activity. For citric acid production, iron limitation is deliberately imposed (iron is a cofactor for aconitase, the enzyme that would otherwise consume citric acid in the TCA cycle).
