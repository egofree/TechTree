# Bar finishing

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Bar finishing (from soap noodles to finished bar) — - **Mixing**: Soap noodles + additives (fragrance, color, titanium dioxide whitener, antibacterial agents, moisturizers) mixed in an amalgamator (screw mixer) at 40-50°C. - **Milling**: Pass soap through 3-5 roller mills (refining mills) to homogenize texture and eliminate air pockets.
