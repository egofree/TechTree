# Minimum feature size

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

The smallest dimension that can be reliably printed and etched. For a given process node (e.g., "7 nm"), the actual minimum metal half-pitch is typically 30-40 nm, contact half-pitch 30-40 nm, and fin width 6-8 nm. These are not arbitrary — they reflect lithography resolution, etch selectivity, and CMP uniformity limits.
