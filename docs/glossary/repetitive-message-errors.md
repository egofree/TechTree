# Repetitive message errors

> **Type**: material | **Tier**: important | **Domains**: transport

Under high-volume traffic (stock exchanges, news wires), operators copying at speed make character errors that distort meaning. A single misplaced dot can change "BUY" to "BUT" or "SEND" to "SOLD". Financial telegraph offices use a verification operator who reads back each message to the sending operator before transmission. Accuracy above 99.5% is the target; at 20 WPM, this means fewer than 1 error per 400 characters.
