# Thermocouple

> **Type**: noun | **Tier**: important | **Domains**: measurement

Two dissimilar metals joined at measurement junction. Voltage generated proportional to temperature difference between measurement junction and reference junction. Type K (chromel/alumel): -200°C to +1300°C, ~41 μV/°C. Type S (Pt/Pt-10%Rh): 0°C to +1600°C, ~6 μV/°C. Read with precision millivoltmeter or potentiometer. Accuracy ±1-2°C with reference junction compensation.
