# Type 304

> **Type**: equipment | **Tier**: important | **Domains**: metals

Type 304 (18Cr-8Ni, the workhorse) — ~18% Cr, ~8% Ni, ≤0.08% C. Excellent corrosion resistance in atmospheric and mildly corrosive environments. Used for food processing, chemical vessels, architectural applications. Tensile strength: 515-620 MPa (annealed), up to 1,200+ MPa (cold-worked). Maximum service temperature: ~870°C (intermittent), ~925°C (continuous, oxidation limit). See: [Specialty Alloys](../metals/alloys.md)
