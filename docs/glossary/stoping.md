# Stoping

> **Type**: material | **Tier**: important | **Domains**: mining

Stoping (ore extraction) — Extract ore between levels. Overhand stoping: work upward from level. Underhand: work downward. Shrinkage: leave broken ore in stope as working platform. Cut-and-fill: backfill with waste as you go (better ground control). See: [Mining Extraction](../mining/extraction.md)
