# Strowger switch

> **Type**: noun | **Tier**: important | **Domains**: computing

(telephone switching): - **Design**: Two-motion selector — vertical (10 levels) then horizontal (10 positions) = 100-contact array per switch. Input pulse train from rotary dial: first digit selects vertical level, second digit selects horizontal position. Routes one telephone line to any of 100 destination lines automatically. - **Application**: Automatic telephone exchanges eliminating human operators.
