# Addition technique

> **Type**: noun | **Tier**: important | **Domains**: computing

Set first addend on the beads. Process the second addend digit by digit, left to right (most significant digit first, opposite to pencil-and-paper right-to-left). For each digit, add the value using a combination of upper and lower bead movements. Example: adding 7 to a rod showing 4: move up one upper bead (+5) and two lower beads (+2), net +7. If the rod exceeds 9, clear it and carry 1 left. Trained operators sustain 10-20 additions per minute on multi-digit numbers.
