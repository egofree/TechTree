# Colorimetric and spectrophotometric methods

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Based on Beer-Lambert law: A = εbc, where A = absorbance, ε = molar absorptivity (L/mol·cm), b = path length (cm), c = concentration (mol/L). Measure absorbance at specific wavelength using UV-visible spectrophotometer. Detection limits: 0.1-10 ppm for most analytes. Common applications: phosphate (molybdenum blue method, 880 nm), iron (phenanthroline method, 510 nm), silica (molybdosilicate method, 410 nm). Atomic absorption spectroscopy (AAS) for trace metals: detection limits 0.01-1 ppm.
