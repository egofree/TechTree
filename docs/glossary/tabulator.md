# Tabulator

> **Type**: noun | **Tier**: important | **Domains**: computing

(IBM 407): Reads cards at 150 per minute. Performs addition and subtraction on up to 20 counters (each up to 8 digits). Prints one line per card or per control group, with selective printing based on punch patterns. Program control via removable wiring panel (plugboard) that defines which columns feed which counters and when printing occurs.
