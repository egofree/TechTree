# Endurance

> **Type**: noun | **Tier**: important | **Domains**: computing

SLC (single-level cell, 1 bit/cell): 50,000-100,000 write cycles. MLC (2 bits/cell): 3,000-10,000 cycles. TLC (3 bits/cell): 1,000-3,000 cycles. QLC (4 bits/cell): 100-1,000 cycles. Wear leveling (distributing writes across all blocks) extends usable life. Error correction (BCH or LDPC codes) handles bit errors that increase with wear.
