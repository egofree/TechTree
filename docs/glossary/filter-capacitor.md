# Filter capacitor

> **Type**: noun | **Tier**: important | **Domains**: silicon

Add a reservoir capacitor across the DC output to smooth the 120 Hz (or 100 Hz) ripple. Size: C ≥ Iload/(2·f·ΔV), where ΔV is the allowable ripple voltage. For 1A load and 2V ripple at 60 Hz: C ≥ 1/(2·60·2) = 4,200 μF. Use a 4,700 μF/25V electrolytic capacitor.
