# Dugout canoe

> **Type**: noun | **Tier**: important | **Domains**: transport

Select straight tree (oak, pine, cedar) 3-10 m long. Hollow by burning controlled fires inside and adzing charred wood. Shape hull — round bottom for speed, flat for stability. Thickness 2-5 cm. Freeboard 20-40 cm. Capacity 0.5-2 tonnes. Oldest boat type, requires only fire and axes.
