# Channeling in packed columns

> **Type**: material | **Tier**: important | **Domains**: chemistry

Liquid maldistribution causes some areas of the packing to run dry (vapor passes through with no liquid contact, no separation) while other areas flood. Causes: inadequate liquid distributor design, damaged distributor, or packing installation not level. Symptoms: poor separation despite adequate packing height, asymmetrical temperature profile. Prevention: liquid distributors with >40 pour points per m² of column cross-section, verified by water distribution test before packing installation....
