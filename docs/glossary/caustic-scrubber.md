# Caustic scrubber

> **Type**: material | **Tier**: important | **Domains**: gas-handling

NaOH solution (5-20%) in packed column. Removes acid gases: Cl₂ (forms NaOCl + NaCl), HCl (forms NaCl), H₂S (forms Na₂S), SO₂ (forms Na₂SO₃). Essential for exhaust gas treatment before venting to atmosphere. Monitor scrubber solution pH (replace when pH drops below 10).
