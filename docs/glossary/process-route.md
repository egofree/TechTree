# Process route

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Process route (from MG-Si, Silicon stage) — 1. **Trichlorosilane synthesis**: MG-Si + 3HCl → SiHCl₃ + H₂ (fluidized bed reactor, 280-350°C, Cu catalyst).
