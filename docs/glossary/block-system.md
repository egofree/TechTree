# Block system

> **Type**: material | **Tier**: important | **Domains**: transport

Divide track into blocks (1-3 km sections). Only one train permitted in each block at a time. Block instruments (telegraph-based) at each block post — operator confirms train has cleared before allowing next entry. Prevents rear-end collisions. Absolute block: next train must wait for clear signal. Permissive block: following train may enter at restricted speed (lower capacity, used on lightly trafficked lines).
