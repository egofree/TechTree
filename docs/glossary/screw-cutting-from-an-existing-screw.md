# Screw-cutting from an existing screw

> **Type**: material | **Tier**: important | **Domains**: machine-tools

Mount an existing screw (even a rough hand-cut one) as a master template. Connect it to the spindle via change gears so the cutting tool advances exactly one pitch per revolution. The master screw's errors are reproduced, but each generation of screw can be made more accurate by using the best available screw as the template.
