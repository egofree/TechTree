# MOSFET threshold voltage

> **Type**: noun | **Tier**: important | **Domains**: silicon

Apply gate voltage sweep from 0 to 5V with drain at 0.1V. Plot Id vs. Vg. Threshold voltage Vth = gate voltage where drain current reaches a specified value (e.g., 1 μA × W/L). Check that Vth is within specification (typically 0.5-2V for enhancement mode).
