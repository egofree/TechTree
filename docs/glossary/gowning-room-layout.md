# Gowning room layout

> **Type**: material | **Tier**: important | **Domains**: photolithography

Garment storage in sealed cabinets with HEPA-filtered air supply. Clean garments dispensed from individually sealed bags. Soiled garment return through a pass-through window to the laundry collection area, preventing cross-contamination. Full-length mirror for self-inspection before entering the cleanroom (verify no exposed skin, no loose hair, proper glove seal).
