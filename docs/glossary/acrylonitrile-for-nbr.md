# Acrylonitrile for NBR

> **Type**: equipment | **Tier**: important | **Domains**: polymers

Flammable liquid (flash point -1°C), toxic by inhalation and skin absorption. IARC Group 2B (possible carcinogen). Cyanide release on combustion — fires involving acrylonitrile produce hydrogen cyanide (HCN) gas. OSHA PEL 2 ppm (8-hour TWA). Handle in closed systems with leak detection. Emergency response for large spills: evacuate 100+ m radius, use self-contained breathing apparatus for approach.
