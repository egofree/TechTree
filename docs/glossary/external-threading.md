# External threading

> **Type**: noun | **Tier**: important | **Domains**: energy, machine-tools

Tool shaped to thread profile (60° for ISO/Unified). Engage half-nut, take successive passes increasing depth 0.1-0.15 mm per pass. Total infeed ≈ 0.614 × pitch for 60° threads. Use thread dial indicator to re-engage on the same groove.
