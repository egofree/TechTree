# Continuous particle counters

> **Type**: noun | **Tier**: important | **Domains**: photolithography

Laser scattering counters installed at critical locations (near process tools, at return air grilles). Count particles ≥0.1-5 μm in real-time. Data logged continuously for trend analysis. Alarms trigger at threshold levels (e.g., >100 particles/ft³ ≥0.5 μm for Class 100).
