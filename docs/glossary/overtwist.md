# Overtwist

> **Type**: material | **Tier**: important | **Domains**: textiles

The yarn kinks back on itself, forming tight corkscrews that cannot be unwound. Cause: more twist was inserted than the yarn diameter can hold without curling. The yarn "wants" to relieve the excess twist energy by wrapping around itself. Fix: reduce wheel speed or increase drafting speed so fewer twists are inserted per unit length. If a section is already overtwisted, run it back through the wheel in the opposite direction to remove some twist before plying.
