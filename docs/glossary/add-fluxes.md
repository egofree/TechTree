# Add fluxes

> **Type**: noun | **Tier**: important | **Domains**: metals

Burned lime (CaO, 30-70 kg/tonne steel) and dolomite (CaO·MgO, 5-15 kg/tonne). Flux creates a basic slag (V-ratio = CaO/SiO₂ = 2.5-4.0) that absorbs SiO₂, MnO, and P₂O₅. Fluorspar (CaF₂, 2-5 kg/tonne) may be added to improve slag fluidity.
