# Clay token numerals

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Clay token numerals (~8000-3500 BCE) — distinct token shapes for distinct commodities and quantities. Spherical token = "10", cone = "1" (small measure of grain). These tokens and their impressions on bullae are the direct ancestors of both writing and abstract numerals. See: [Writing & Record-Keeping](../knowledge/writing.md)
