# NAND flash

> **Type**: noun | **Tier**: important | **Domains**: computing

Page-level reads (page = 2-16 KB). Faster erase and write than NOR. Much higher density (smaller cells, no random-access decoding overhead). Used for mass storage (SSDs, USB drives, memory cards). A 3D NAND chip stacks 100+ layers vertically, achieving 1 Tb per die.
