# Design for manufacturing (DFM)

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Beyond DRC, DFM optimizes layout for maximum yield. Critical area analysis identifies locations where a random defect of given size would cause a short or open — layouts are modified to reduce critical area. Via redundancy (double vias where routing allows) prevents single-via opens. Wire spreading increases spacing between long parallel runs to reduce the probability of particle-caused shorts.
