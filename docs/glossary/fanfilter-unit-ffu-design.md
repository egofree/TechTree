# Fan/filter unit (FFU) design

> **Type**: noun | **Tier**: important | **Domains**: photolithography

Integrated module — centrifugal fan motor + HEPA filter + pre-filter in a single ceiling tile-sized panel (2×2 ft or 2×4 ft standard). Speed control (adjustable fan RPM for desired air velocity, typically 0.3-0.5 m/s). Modular installation: mount in cleanroom ceiling grid, wire to power, adjust speed. Enables incremental cleanroom expansion.
