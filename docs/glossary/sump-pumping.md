# Sump pumping

> **Type**: material | **Tier**: important | **Domains**: mining

At the lowest point of the mine (the shaft bottom), excavate a sump (collection basin, 2-3 m deep, lined with concrete or brick). Install a pump intake in the sump. The sump provides buffer capacity — if pump flow exceeds inflow, the sump fills; if inflow exceeds pumping, the sump provides warning time before flooding. Sump capacity should hold at least 2-4 hours of normal water inflow to allow time for pump maintenance without flooding.
