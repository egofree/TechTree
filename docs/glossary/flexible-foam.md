# Flexible foam

> **Type**: material | **Tier**: important | **Domains**: polymers

Flexible foam (mattresses, cushions, upholstery) — polyether polyol (MW ~3000, 100 parts — the high MW gives long, flexible chain segments between cross-links) + TDI (50-60 parts — index 105, meaning 5% excess isocyanate for hard segment formation) + water (3-5 parts — generates CO₂ blowing agent via the reaction H₂O + NCO → urea linkage + CO₂; the CO₂ inflates the foam.
