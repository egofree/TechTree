# Baudot code

> **Type**: noun | **Tier**: important | **Domains**: computing

Baudot code (5-bit) — 32 characters. Two shift states (letters shift and figures shift) double the effective character set to ~60. Each character: 1 start bit (space), 5 data bits, 1.42 stop bits (mark). At 45.5 baud (standard teletype speed), one character takes 163 ms (6 characters/second). Synchronous motor drives both sender and receiver at the same speed. See: [Electromechanical Computing](../computing/electromechanical.md)
