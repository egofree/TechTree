# Dow process

> **Type**: noun | **Tier**: important | **Domains**: metals

Dehydrated MgCl₂·1.5H₂O fed directly to cell (partially hydrated — generates some HCl, managed by cell design). Electrolyte: NaCl-KCl-MgCl₂ melt at 700-750°C. Steel cathode, graphite anode. Mg²⁺ + 2e⁻ → Mg(l) (rises to surface, protected by inert gas blanket). 2Cl⁻ → Cl₂(g). Cell voltage 5-7 V, current 50-100 kA. Energy: 16-18 kWh/kg Mg.
