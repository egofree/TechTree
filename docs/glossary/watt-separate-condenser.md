# Watt separate condenser

> **Type**: noun | **Tier**: important | **Domains**: energy

(~1769): - Cylinder stays hot; separate condenser stays cold. Eliminates the wasteful heating/cooling cycle. Double-acting (steam on both sides of piston). Sun-and-planet gear provides rotary output. Centrifugal governor regulates speed. ~3-5% efficiency. 10-100+ HP. Enabled factory power via line shaft.
