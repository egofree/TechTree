# Multiplication procedure

> **Type**: noun | **Tier**: important | **Domains**: computing

To multiply 456 × 78, set the input sliders to 4-5-6. Turn the crank 8 times (units digit of multiplier). The carriage shifts one position left. Turn the crank 7 times (tens digit). The accumulator now reads 35,568. Total time: ~15 seconds for an experienced operator.
