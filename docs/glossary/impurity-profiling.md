# Impurity profiling

> **Type**: material | **Tier**: important | **Domains**: chemistry

Related substances (degradation products and process impurities) quantified by HPLC with gradient elution. Each known impurity has an individual specification (typically 0.1-0.5% of the active peak area). Total impurities: typically <2.0%. Unknown impurities: each <0.10%. ICH Q3B(R2) sets qualification thresholds for impurities above which toxicological assessment is required.
