# Comparison standard

> **Type**: noun | **Tier**: important | **Domains**: measurement

When a profilometer is unavailable, surface finish can be estimated by comparison with tactile reference specimens — a set of metal blocks with known Ra values produced by different machining processes. Run a fingernail across the workpiece and the standards; match the "feel." Accuracy ±one grade (roughly ±50% of the Ra value) — adequate for most workshop purposes.
