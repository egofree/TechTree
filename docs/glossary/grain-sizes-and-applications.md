# Grain sizes and applications

> **Type**: noun | **Tier**: supporting | **Domains**: mining

Classification of black powder by grain size for different applications: meal powder (finest, for priming), Fg (1-2 mm, for rifles), FFg (0.6-1 mm, general purpose), FFFg (0.3-0.6 mm, for pistols), and C-grain (4-8 mm coarse, for blasting). Grain size affects burn rate and application suitability.
