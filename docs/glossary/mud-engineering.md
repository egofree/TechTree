# Mud engineering

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Bentonite clay (4-8% by weight in water) provides viscosity. Barite (BaSO₄) added for density control (mud must exceed formation fluid pressure). pH controlled with NaOH (9.5-10.5). Mud weight typically 1.0-2.0 g/cm³.
