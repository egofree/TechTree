# Carry propagation

> **Type**: material | **Tier**: important | **Domains**: computing

The most demanding mechanical problem. Babbage's solution uses a vertical stack of 31 digit wheels per column. Each wheel has a projecting tooth that, on passing 9→0, lifts a spring-loaded lever that advances the wheel above by one position. Carries propagate upward (ones to tens to hundreds) during a single crank rotation. The timing must allow each carry to complete before the next triggers.
