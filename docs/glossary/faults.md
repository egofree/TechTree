# Faults

> **Type**: material | **Tier**: important | **Domains**: mining

Fractures where rock has moved. Fault zones create pathways for hydrothermal fluids and often host vein-type deposits (gold, silver, copper). Map fault traces on surface — linear valleys, offset ridges, aligned springs. Mineralization commonly occurs where faults intersect or change direction. Fault displacement of 1-50 m is common in mining districts.
