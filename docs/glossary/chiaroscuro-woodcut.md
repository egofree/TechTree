# Chiaroscuro woodcut

> **Type**: noun | **Tier**: important | **Domains**: knowledge

Chiaroscuro woodcut (Ugo da Carpi, ~1516) — use multiple blocks, each printing a different tone of the same color (light, medium, dark). Allows tonal gradation without hatching. Each block must be registered to the others. See: [Paper and Printing Press](../knowledge/printing.md)
