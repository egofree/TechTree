# Gas flare system

> **Type**: noun | **Tier**: important | **Domains**: chemistry

A properly designed flare is essential for safe gasifier operation during startup, shutdown, and emergencies. The flare must burn all producer gas whenever it cannot be used by the engine. Stack height: minimum 3 m above the gasifier roofline, with spark arrestor. Flare ignition: continuous pilot flame or automatic spark igniter. Flame monitoring: UV sensor confirms pilot flame is lit.
