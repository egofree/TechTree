# Fire control application

> **Type**: material | **Tier**: important | **Domains**: computing

US Navy Mark 1 through Mark 9 fire control computers used arrays of ball-and-disk integrators. Inputs: target bearing rate, range rate, wind speed, ship velocity components, projectile ballistic data (stored as cam profiles). Each integrator computed one term of the fire control equations. Output shafts drove gun elevation and train indicators directly. The Mark 1 contained about 3,000 precision parts and weighed 1,360 kg.
