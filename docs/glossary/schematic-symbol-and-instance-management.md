# Schematic symbol and instance management

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Each cell has a graphical symbol for schematic entry. Instances reference the master cell — changes propagate to all instances. Parameterized cells (p-cells) allow instance-specific customization (e.g., transistor width). Version control tracks design revisions; release management ensures only validated cell versions are used in production designs.
