# Number storage

> **Type**: material | **Tier**: important | **Domains**: computing

Number storage refers to 23-digit decimal numbers with sign. Stored in 60 sets of 24-position mechanical switches (constant storage, set by hand). Working registers: 72 adding storage units (mechanical counter wheels, each holding a 23-digit number). The switch-set constants could be changed during setup but not during program execution. See: [Electromechanical Computing](../computing/electromechanical.md)
