# Resolution enhancement techniques (RET)

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Umbrella term for OPC, PSM, off-axis illumination (OAI), and sub-resolution assist features (SRAFs). SRAFs are small non-printing features added to the mask that diffract light to improve image contrast at main feature edges. RET is essential for printing features below k₁ = 0.5 (where k₁ = feature size × NA / λ — the normalized resolution parameter).
