# Floorplanning

> **Type**: noun | **Tier**: important | **Domains**: vlsi-scaling

Assign blocks to physical regions on the die. Critical factors: block area, aspect ratio, interconnect congestion between blocks, power grid routing, and signal I/O placement. Macro blocks (SRAM, analog IP) placed first, then standard cell rows fill remaining space. A good floorplan minimizes total wirelength (reduces delay and power).
