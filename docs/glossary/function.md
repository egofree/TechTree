# Function

Function describes the intended purpose and operating principle of a component, mechanism, or system. Understanding function is central to both design and troubleshooting: a component's function determines its material requirements, dimensional tolerances, and failure modes. In photolithography, the pellicle membrane functions by positioning any contaminant particle 5–10 mm above the mask plane—well outside the depth of focus of the projection optics. The particle is blurred to invisibility on the wafer image and does not print as a defect.

In mechanical engineering, the function of each part in an assembly dictates its design parameters. Bearings must support loads with minimal friction; gears must transmit torque at prescribed ratios; seals must exclude contaminants while containing fluids. The function-first design approach identifies what each component must do before determining how to achieve it—a critical discipline in bootstrap engineering where materials and manufacturing options are constrained.

Functional analysis also drives maintenance decisions: components whose failure would be catastrophic (structural members, braking systems) require more conservative design margins and more frequent inspection than those whose failure is merely inconvenient. Documenting the intended function of each component in a bootstrap industrial system provides the foundation for rational maintenance scheduling and spare parts prioritization.
