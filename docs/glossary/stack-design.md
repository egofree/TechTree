# Stack design

> **Type**: noun | **Tier**: important | **Domains**: chemistry

Stack design refers to 50-200 cells in filter-press configuration (bipolar plates between cells). Active area per cell: 0.5-4 m². Total stack: 100-400V DC input, 2-20 kA. Production rate: ~0.4-0.5 Nm³ H₂ per kWh. See: [Electrolysis](../chemistry/electrolysis.md)
