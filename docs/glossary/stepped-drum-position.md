# Stepped drum position

> **Type**: noun | **Tier**: important | **Domains**: computing

In the Leibniz wheel, each digit position has a drum with teeth of length 1 through 9. A sliding gear engages a selectable number of teeth. The axial position of the sliding gear determines how many units are added per drum revolution. This converts a linear input (lever position) to a rotary addition step, enabling direct multiplication through repeated addition.
