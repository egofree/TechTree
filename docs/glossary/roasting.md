# Roasting

> **Type**: material | **Tier**: important | **Domains**: mining

Sulfide ores must be roasted before smelting to remove sulfur and convert metal sulfides to oxides. Heat crushed ore to 500-700°C in air. Sulfur burns off as SO₂ gas (capture for sulfuric acid production if possible — SO₂ + H₂O + catalyst → H₂SO₄). Roasting reactions: 2ZnS + 3O₂ → 2ZnO + 2SO₂, 2PbS + 3O₂ → 2PbO + 2SO₂.
