## Technical Details

Application documentation connects material properties to end uses through engineering reasoning. Nitrile rubber (NBR) exemplifies this: its acrylonitrile (ACN) content determines chemical resistance. High-ACN grades resist fuels and oils, making them suitable for fuel lines, filler neck hoses, and oil seals. Medium-ACN grades balance chemical resistance with low-temperature flexibility, serving as O-rings and rotating shaft seals. Low-ACN grades offer better flexibility for general-purpose gaskets and diaphragms.

The pattern repeats across materials. Tool steel applications map hardness and red-hardness to cutting, forming, and stamping uses. Optical glass applications map refractive index and dispersion to lens, prism, and window uses. Applications sections are structured as property-to-use mappings, enabling engineers to select materials by working backward from the application requirement to the material specification.

For bootstrapping, the applications catalog serves as a menu of capabilities unlocked at each tech-tree stage. Each new material or process expands the set of achievable applications, which in turn enables subsequent capabilities in a chain of dependencies.
