# Angle gauge blocks

> **Type**: noun | **Tier**: important | **Domains**: measurement

A 16-piece set (four blocks each from series 1°, 3°, 5°, 15° plus fractional blocks of 1', 3', 5', 15', 30' and 1°) can be wrung together to build any angle from 0° to 99° in increments of 1 arcsecond. Like length gauge blocks, they adhere by wringing and provide a direct, traceable angle reference without computation.
