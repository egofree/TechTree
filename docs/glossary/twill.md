# Twill

> **Type**: material | **Tier**: important | **Domains**: textiles

Over 2, under 1 (or 2/2 twill). Diagonal rib pattern visible on fabric surface. Drapes better, more flexible. For clothing, work garments. 4 shafts for basic twills; point twills and herringbone use 4-8 shafts. Heddle lifting sequence advances one shaft each pick, creating the diagonal.
