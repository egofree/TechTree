# Sterilization validation

> **Type**: equipment | **Tier**: important | **Domains**: chemistry

Each batch of medium must be sterile before inoculation. Validation: biological indicators (spores of Geobacillus stearothermophilus, D₁₂₁ value = 1.5-3.0 minutes) placed in the medium before sterilization, recovered and cultured after. If no growth, sterilization is confirmed. Thermal death time for typical contaminants: E. coli killed at 72°C for 15 seconds (pasteurization), bacterial spores require 121°C for 15 minutes.
